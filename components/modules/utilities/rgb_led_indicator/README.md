# ­ЪњА рЃЏрЃЮрЃЊрЃБрЃџрЃў: `rgb_led_indicator`

**­ЪЌѓ№ИЈ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ:** `utilities`  
**Рюњ№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃў:** Synapse Framework Team  
**­Ъћќ рЃЋрЃћрЃарЃАрЃўрЃљ:** 2.1.0

## 1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

`rgb_led_indicator` рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃљрЃџрЃБрЃа, рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА RGB LED рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ. рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃўрЃА рЃарЃЮрЃџрЃўрЃА рЃерЃћрЃАрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃџрЃљрЃЊ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЋрЃбрЃЮрЃюрЃЮрЃЏрЃўрЃБрЃарЃљрЃЊ рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃА рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџ рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃќрЃћ рЃЊрЃљ, рЃљрЃЏрЃљрЃЋрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ, рЃљрЃгрЃЋрЃЊрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА API-рЃА рЃЊрЃљ CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃћрЃА рЃЋрЃћрЃарЃАрЃўрЃљ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ **рЃфрЃўрЃцрЃарЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃърЃўрЃюрЃћрЃЉрЃўрЃА** рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ, рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃљрЃЊ рЃЎрЃў **`sn74hc595n_writer`** (shift register) рЃАрЃћрЃарЃЋрЃўрЃАрЃЌрЃљрЃю рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃЊ. рЃЏрЃЮрЃЊрЃБрЃџрЃА рЃљрЃЦрЃЋрЃА рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃЊрЃљ рЃфрЃўрЃЏрЃфрЃўрЃЏрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ рЃфрЃљрЃџрЃЎрЃћ FreeRTOS рЃбрЃљрЃАрЃЎрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ, рЃарЃљрЃЌрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃџрЃў рЃўрЃДрЃЮрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃљрЃарЃљрЃЉрЃџрЃЮрЃЎрЃўрЃарЃћрЃЉрЃљрЃЊрЃў, рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ.

## 2. ­ЪЈЌ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

- **рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃљ:** рЃћрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃА рЃЎрЃџрЃљрЃАрЃўрЃЎрЃБрЃарЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў. рЃўрЃА рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃљрЃарЃўрЃА:
  - **Service Provider:** рЃљрЃгрЃЋрЃЊрЃўрЃА рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА `rgb_led_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА.
  - **Service Consumer:** рЃЌрЃљрЃЋрЃљрЃЊ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА `sn74hc595n_writer_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃърЃўрЃюрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ.
- **рЃ░рЃўрЃЉрЃарЃўрЃЊрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃљрЃарЃўрЃА **Service-driven** рЃЊрЃљ **Event-driven**.
  - рЃўрЃА **рЃљрЃгрЃЋрЃЊрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА** рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃћрЃа рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА.
  - рЃўрЃА **рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА** `Event Bus`-рЃќрЃћ, рЃарЃљрЃЌрЃљ рЃљрЃАрЃљрЃ«рЃЮрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃќрЃЮрЃњрЃљрЃЊрЃў рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ.
- **рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљрЃќрЃћ-рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ:** `start` рЃцрЃљрЃќрЃљрЃерЃў, рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА (рЃЏрЃљрЃњ., `ble_provisioning`) рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃАрЃбрЃљрЃбрЃБрЃАрЃА рЃЊрЃљ рЃЊрЃљрЃБрЃДрЃЮрЃЋрЃюрЃћрЃЉрЃџрЃўрЃЋ рЃљрЃДрЃћрЃюрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃфрЃўрЃљрЃА.
- **рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў:** LED рЃћрЃцрЃћрЃЦрЃбрЃћрЃЉрЃў рЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃў рЃбрЃљрЃАрЃЎрЃўрЃА рЃЏрЃўрЃћрЃа, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ API рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃћрЃЉрЃў рЃљрЃа рЃўрЃДрЃЮрЃА рЃЏрЃЉрЃџрЃЮрЃЎрЃљрЃЋрЃў.
- **рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ:** рЃЏрЃћрЃЦрЃљрЃюрЃўрЃЎрЃБрЃа рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА (API/CLI рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃЌ) рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃБрЃЏрЃљрЃдрЃџрЃћрЃАрЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃў рЃљрЃЦрЃЋрЃА. `release_control` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃа, рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў.
- **CLI рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА `led` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА `Command Router`-рЃерЃў.

## 3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ (`config.json`)

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃАрЃарЃБрЃџрЃљрЃЊ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџ shift register-рЃА рЃЊрЃљ рЃарЃЮрЃЏрЃћрЃџ рЃърЃўрЃюрЃћрЃЉрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃўрЃА.

**рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

```json
[
  {
    "type": "rgb_led_indicator",
    "enabled": true,
    "config": {
        "instance_name": "status_led",
        "sr_writer_service": "main_sr_writer",
        "red_pin": 4,
        "green_pin": 5,
        "blue_pin": 6,
        "is_common_anode": false
    }
  }
]
```

**рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

| рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў | рЃбрЃўрЃърЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ |
| :--- | :--- | :--- | :---: |
| `instance_name` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў. | РюЁ |
| `sr_writer_service` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃўрЃЏ `sn74hc595n_writer` рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА `instance_name`, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф LED рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ. | РюЁ |
| `red_pin` | рЃарЃўрЃфрЃ«рЃЋрЃў | RED рЃљрЃарЃ«рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў рЃърЃўрЃюрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў shift register-рЃќрЃћ (0-7, 8-15 рЃЊрЃљ рЃљ.рЃе.). | РюЁ |
| `green_pin` | рЃарЃўрЃфрЃ«рЃЋрЃў | GREEN рЃљрЃарЃ«рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў рЃърЃўрЃюрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў shift register-рЃќрЃћ. | РюЁ |
| `blue_pin` | рЃарЃўрЃфрЃ«рЃЋрЃў | BLUE рЃљрЃарЃ«рЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў рЃърЃўрЃюрЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў shift register-рЃќрЃћ. | РюЁ |
| `is_common_anode` | boolean | `true`, рЃЌрЃБ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃљрЃюрЃЮрЃЊрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ RGB LED. | РЮї (Default: `false`) |

## 4. ­ЪЏа№ИЈ Service API (`rgb_led_api_t`)

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃарЃЮрЃЏрЃџрЃўрЃА рЃбрЃўрЃърЃўрЃљ `FMW_SERVICE_TYPE_RGB_LED_API`.

**API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:**

- **`esp_err_t set_color(uint8_t r, uint8_t g, uint8_t b);`**  
    рЃљрЃДрЃћрЃюрЃћрЃЉрЃА рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃа рЃцрЃћрЃарЃА. рЃарЃљрЃЊрЃњрЃљрЃю рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃфрЃўрЃцрЃарЃБрЃџрЃўрЃљ, рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ >127 рЃљрЃдрЃўрЃЦрЃЏрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф ON, рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃюрЃў - OFF.

- **`esp_err_t start_blink(uint8_t r, uint8_t g, uint8_t b, uint32_t interval_ms);`**  
    рЃўрЃгрЃДрЃћрЃЉрЃА рЃфрЃўрЃЏрЃфрЃўрЃЏрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў рЃцрЃћрЃарЃўрЃЌрЃљ рЃЊрЃљ рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃўрЃЌ.

- **`esp_err_t start_pulse(uint8_t r, uint8_t g, uint8_t b, uint32_t period_ms);`**  
    **(рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ)** рЃфрЃўрЃцрЃарЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮ, рЃћрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА **рЃюрЃћрЃџ рЃфрЃўрЃЏрЃфрЃўрЃЏрЃА** рЃЊрЃљ рЃљрЃарЃљ рЃњрЃџрЃБрЃЋ рЃърЃБрЃџрЃАрЃљрЃфрЃўрЃљрЃА.

- **`esp_err_t turn_off();`**  
    рЃЌрЃўрЃерЃљрЃЋрЃА LED рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃА.

- **`esp_err_t release_control();`**  
    рЃљрЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃћрЃЉрЃА "рЃЏрЃћрЃЦрЃљрЃюрЃўрЃЎрЃБрЃа" рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА рЃЊрЃљ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў.

## 5. Рїе№ИЈ CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў (Command Line Interface)

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА `led` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА `Command Router`-рЃерЃў.

| рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ | рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
|:---|:---|:---|
| `led set` | `<r> <g> <b>` | ­Ъје рЃљрЃДрЃћрЃюрЃћрЃЉрЃА рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃа рЃцрЃћрЃарЃА. рЃЏрЃљрЃњ: `led set 255 0 0` (рЃгрЃўрЃЌрЃћрЃџрЃў). |
| `led blink` | `<r> <g> <b> <interval>` | Рюе рЃўрЃгрЃДрЃћрЃЉрЃА рЃфрЃўрЃЏрЃфрЃўрЃЏрЃА. `<interval>` рЃљрЃарЃўрЃА рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃћрЃЉрЃерЃў. |
| `led pulse` | `<r> <g> <b> <period>` | ­Ъїі рЃўрЃгрЃДрЃћрЃЉрЃА рЃюрЃћрЃџ рЃфрЃўрЃЏрЃфрЃўрЃЏрЃА. `<period>` рЃљрЃарЃўрЃА рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃћрЃЉрЃерЃў. |
| `led off` | - | РџФ рЃЌрЃўрЃерЃљрЃЋрЃА LED-рЃА. |
| `led release` | - | ­Ъћё рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА LED-рЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў. |

## 6. ­ЪЊб рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃАрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ:

| рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў | LED-рЃўрЃА рЃЦрЃфрЃћрЃЋрЃљ |
|:---|:---|
| `WIFI_CREDENTIALS_NOT_FOUND` | ­ЪњЏ рЃюрЃћрЃџрЃў рЃДрЃЋрЃўрЃЌрЃћрЃџрЃў рЃфрЃўрЃЏрЃфрЃўрЃЏрЃў (Provisioning) |
| `WIFI_EVENT_CONNECTED` | ­Ъњџ рЃЏрЃгрЃЋрЃљрЃюрЃћрЃЊ рЃљрЃфрЃўрЃЏрЃфрЃўрЃЏрЃћрЃЉрЃљ 3-рЃ»рЃћрЃа, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЌрЃўрЃерЃЋрЃљ |
| `WIFI_EVENT_DISCONNECTED` | РЮц№ИЈ рЃБрЃгрЃДрЃЋрЃћрЃбрЃў рЃгрЃўрЃЌрЃћрЃџрЃў рЃцрЃћрЃарЃў |
| `SYSTEM_HEALTH_ALERT` | РЮц№ИЈ рЃАрЃгрЃарЃљрЃцрЃў рЃгрЃўрЃЌрЃћрЃџрЃў рЃфрЃўрЃЏрЃфрЃўрЃЏрЃў |

## 7. РЮЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў

- **рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ `CMakeLists.txt`-рЃерЃў:** `rgb_led_indicator`-рЃўрЃА `CMakeLists.txt` рЃцрЃљрЃўрЃџрЃерЃў, `REQUIRES` рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў `interfaces`.
- **`init_level`:** рЃљрЃЏ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА `init_level` (`60`) рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃцрЃарЃЮ рЃЏрЃљрЃдрЃљрЃџрЃў, рЃЋрЃўрЃЊрЃарЃћ рЃўрЃЏ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА (`command_router` - `50`, `sn74hc595n_writer` - `15`).
