Public Service API contract for the System Manager.

Author
Synapse Framework Team
Version

Date

This header defines the abstract interface (API structure) for the System Manager service. Any module wishing to interact with core system functionalities (like getting the module list) via the Service Locator should include this file. This ensures that the consumer module is decoupled from the specific implementation of the System Manager.
