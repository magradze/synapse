Detailed Description

The public Service API structure for the Security Status Reporter.

This is the "contract" that the security_status_reporter module implements and other modules (like self_test_manager) consume.

bool(* security_status_api_t::is_flash_encryption_enabled) (void)

Checks if Flash Encryption is enabled on the device.

Similar to the secure boot check, this returns a cached value. It allows other modules to confirm that the device's flash memory is encrypted, protecting the firmware and data at rest.

Returns
  • true: If Flash Encryption is enabled.
  • false: If Flash Encryption is disabled or the service is not available.

◆ is_secure_boot_enabled

bool(* security_status_api_t::is_secure_boot_enabled) (void)

Checks if Secure Boot is enabled on the device.

This function returns a cached value that is read once at system startup. It provides a quick and efficient way to verify if the device is running in a secure boot environment.

Returns
  • true: If Secure Boot is enabled.
  • false: If Secure Boot is disabled or the service is not available.
