Detailed Description

მოდულის ძირითადი სტრუქტურა, რომელიც განსაზღვრავს ფრეიმვორქის მოდულის კონტრაქტს.

ეს სტრუქტურა შეიცავს მოდულის ყველა აუცილებელ ინფორმაციას და ფუნქციონალურს: სახელს, სტატუსს, კონფიგურაციას, API ფუნქციებს და შიდა მონაცემებს. ყველა მოდული უნდა აღებულ იქნას ამ სტრუქტურის საფუძველზე.
