Detailed Description

Internal shared definitions for the UI Manager module.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-09-12

This private header file is shared ONLY among the .c files within the ui_manager module. It defines the internal data structures, such as the component registry node and the extended private_data struct, ensuring that all parts of the module have a consistent view of the internal state. It MUST NOT be included by any file outside of this module.
