Detailed Description

Declares the rendering functions for all UI screens and common components.

This file contains the function prototypes for each distinct screen (like the home screen and splash screen) and reusable UI components (like the header and footer buttons). These functions are responsible for all drawing operations and are called by the main UI orchestrator.

Formats a module name string.

Formats a module name for display.

This function takes an original module name string and formats it according to predefined rules, storing the result in the provided buffer. The formatted string will not exceed the specified buffer size, including the null terminator.

Parameters
[in]originalPointer to the original module name string to be formatted.
[out]formattedPointer to the buffer where the formatted string will be stored.
[in]sizeSize of the formatted buffer in bytes, including the null terminator.
Note
The function ensures that the formatted string is null-terminated.
If the original string is longer than (size - 1), it will be truncated.
Warning
The formatted buffer must be large enough to hold the formatted string and the null terminator.
