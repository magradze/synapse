## 1. ­Ъћї Service API (`security_status_api_t`)

рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃю рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃљрЃЏ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ `Service Locator`-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ, `SYNAPSE_SERVICE_TYPE_SECURITY_API` рЃбрЃўрЃърЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

**рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃ░рЃћрЃЊрЃћрЃарЃў:** `security_status_interface.h`

### API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

-   **`bool is_secure_boot_enabled(void);`**
    -   **рЃљрЃдрЃгрЃћрЃарЃљ:** рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ рЃЌрЃБ рЃљрЃарЃљ Secure Boot.
    -   **рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА:** `true` рЃЌрЃБ рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ, `false` рЃЌрЃБ рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃўрЃљ.

-   **`bool is_flash_encryption_enabled(void);`**
    -   **рЃљрЃдрЃгрЃћрЃарЃљ:** рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃўрЃљ рЃЌрЃБ рЃљрЃарЃљ flash рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ.
    -   **рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА:** `true` рЃЌрЃБ рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃўрЃљ, `false` рЃЌрЃБ рЃљрЃарЃљ.

**рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў (`selftest_manager`-рЃерЃў):**

```c
service_handle_t sec_handle = synapse_service_lookup_by_type(SYNAPSE_SERVICE_TYPE_SECURITY_API);
if (sec_handle) {
    security_status_api_t *sec_api = (security_status_api_t *)sec_handle;
    bool sb_enabled = sec_api->is_secure_boot_enabled();
    printf("Secure Boot status: %s\n", sb_enabled ? "ENABLED" : "DISABLED");
}
```
