Detailed Description

Implementation of the Relay Actuator module.

Author
Synapse Framework Team
Version
1.3.0
Date
2025-07-19

This module provides a standardized interface to control a relay. It is controlled via a CLI command and publishes state change events to the Event Bus for system-wide status updates. It uses the Resource Manager to lock its GPIO pin and the Storage Manager to persist its state if configured to do so. This version is simplified and does not expose a direct Service API, as control is centralized through the Command Router.
