рЃЏрЃЮрЃЊрЃБрЃџрЃў: Health Monitor

1. рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў: health_monitor
рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ: system
рЃЋрЃћрЃарЃАрЃўрЃљ: 1.0.0
рЃљрЃЋрЃбрЃЮрЃарЃў: Synapse Framework Team

Health Monitor рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃАрЃљрЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃЮрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЏрЃЊрЃњрЃарЃљрЃЊрЃЮрЃЉрЃўрЃА, рЃАрЃљрЃўрЃЏрЃћрЃЊрЃЮрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЌрЃЋрЃўрЃЌрЃЊрЃўрЃљрЃњрЃюрЃЮрЃАрЃбрЃўрЃЎрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃћрЃЉрЃА рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў. рЃўрЃА рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃљрЃЎрЃЋрЃўрЃарЃЊрЃћрЃЉрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃбрЃљрЃАрЃЎрЃћрЃЉрЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ, рЃљрЃдрЃЏрЃЮрЃљрЃЕрЃћрЃюрЃА рЃљрЃюрЃЮрЃЏрЃљрЃџрЃўрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃ«рЃЊрЃћрЃюрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљрЃА рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃўрЃЌ.

2. рЃгрЃўрЃюрЃљрЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃў рЃЊрЃљ <tt>menuconfig</tt>

рЃљрЃЏ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃбрЃљрЃАрЃЎрЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ ESP-IDF-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃерЃў (menuconfig) рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЮрЃцрЃфрЃўрЃћрЃЉрЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ:

  1. рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ idf.py menuconfig.
  2. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ Component config —> FreeRTOS —> Kernel.
  3. рЃЕрЃљрЃарЃЌрЃћрЃЌ ([y]) рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЮрЃарЃў рЃЮрЃцрЃфрЃўрЃљ:
    • Enable FreeRTOS-specific stats formatting
    • Enable generation of run time stats

рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ: рЃљрЃЏ рЃЮрЃцрЃфрЃўрЃћрЃЉрЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, uxTaskGetSystemState рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ undefined reference рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃЌ.

3. рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

  • рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃў рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў: рЃЏрЃЮрЃЊрЃБрЃџрЃА рЃљрЃЦрЃЋрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў FreeRTOS рЃбрЃљрЃАрЃЎрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃўрЃЌ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА "рЃърЃБрЃџрЃАрЃА".
  • рЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў (Thresholds): рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃЦрЃЋрЃћрЃе рЃЏрЃДрЃЮрЃц рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃА рЃљрЃЦрЃЋрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.
  • рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљ: рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃўрЃАрЃљрЃА, рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА SYSTEM_HEALTH_ALERT рЃўрЃЋрЃћрЃюрЃЌрЃА Event Bus-рЃќрЃћ.
  • рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ: рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃЮрЃю рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў "рЃ»рЃљрЃюрЃЏрЃарЃЌрЃћрЃџрЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃўрЃА" рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў.
  • Service-driven: рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃгрЃЋрЃЊрЃўрЃА health_api_t рЃАрЃћрЃарЃЋрЃўрЃАрЃА Service Locator-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ.

4. рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ system_config.json рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

{
"type": "health_monitor",
"enabled": true,
"config": {
"instance_name": "main_health_monitor",
"check_interval_sec": 60,
"thresholds": {
"min_free_heap_kb": 40,
"task_stack_watermark_percent": 90
}
}
}

рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:

рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃбрЃўрЃърЃў рЃљрЃдрЃгрЃћрЃарЃљ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ Default
instance_name рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў. РЮї main_health_monitor
check_interval_sec рЃарЃўрЃфрЃ«рЃЋрЃў рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃў рЃгрЃљрЃЏрЃћрЃЉрЃерЃў. РЮї 30
thresholds рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў. РЮї -
thresholds.min_free_heap_kb рЃарЃўрЃфрЃ«рЃЋрЃў рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃљрЃџрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃќрЃдрЃЋрЃљрЃарЃў рЃЎрЃўрЃџрЃЮрЃЉрЃљрЃўрЃбрЃћрЃЉрЃерЃў. РЮї 50
thresholds.task_stack_watermark_percent рЃарЃўрЃфрЃ«рЃЋрЃў рЃбрЃљрЃАрЃЎрЃўрЃА рЃАрЃбрЃћрЃЎрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃќрЃдрЃЋрЃљрЃарЃў рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃћрЃЉрЃерЃў. РЮї 85

5. Service API (<tt>health_api_t</tt>)

рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃю рЃгрЃЋрЃЊрЃЮрЃЏрЃљ Health Monitor-рЃўрЃА API-рЃќрЃћ Service Locator-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ. рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃбрЃўрЃърЃў: FMW_SERVICE_TYPE_HEALTH_API.

API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:

  • **esp_err_t get_system_health_report(cJSON **report);**
