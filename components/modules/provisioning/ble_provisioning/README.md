# ­ЪЊ▓ рЃЏрЃЮрЃЊрЃБрЃџрЃў `ble_provisioning`

## 1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

**рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў:** `ble_provisioning`  
**рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ:** `provisioning`  
**рЃЋрЃћрЃарЃАрЃўрЃљ:** 2.3.1  
**рЃљрЃЋрЃбрЃЮрЃарЃў:** Synapse Framework Team

`ble_provisioning` рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА (provisioning) рЃърЃарЃЮрЃфрЃћрЃАрЃА **Bluetooth Low Energy (BLE)**-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА ESP-IDF-рЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ `wifi_provisioning` рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃА, рЃарЃљрЃЌрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃЏ рЃерЃћрЃФрЃџрЃЮрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃЊ рЃЏрЃўрЃўрЃдрЃЮрЃА WiFi рЃЦрЃАрЃћрЃџрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў (SSID рЃЊрЃљ рЃърЃљрЃарЃЮрЃџрЃў) рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃЊрЃљрЃю (рЃЏрЃљрЃњ., Espressif-рЃўрЃА **ESP BLE Prov**).

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃгрЃДрЃћрЃЉрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА, рЃЌрЃБ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў, рЃЊрЃљ рЃљрЃАрЃћрЃЋрЃћ, рЃЏрЃўрЃАрЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃљрЃЊ, рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃћрЃа.

## 2. ­ЪЈЌ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

-   **рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў (Service-driven):** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА `ble_prov_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА `Service Locator`-рЃерЃў. рЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА, рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃю, рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў, рЃљрЃю рЃЊрЃљрЃўрЃгрЃДрЃЮрЃю рЃўрЃА рЃўрЃФрЃБрЃџрЃћрЃЉрЃўрЃЌ.
-   **рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃќрЃћ-рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў (Event-driven):** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃЦрЃбрЃўрЃБрЃарЃљрЃЊ рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА `Event Bus`-рЃЌрЃљрЃю:
    -   **рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА** рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ `PROV_STARTED`, `PROV_ENDED` рЃЊрЃљ `PROV_CREDENTIALS_RECEIVED`, рЃарЃљрЃЌрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃАрЃ«рЃЋрЃљ рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃЏрЃљ рЃўрЃфрЃЮрЃЊрЃюрЃћрЃю рЃЏрЃўрЃАрЃў рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.
    -   **рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА** рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ `WIFI_EVENT_IP_ASSIGNED` (рЃарЃљрЃЌрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃљрЃАрЃарЃБрЃџрЃЮрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў) рЃЊрЃљ `PROV_START_REQUESTED` (рЃарЃљрЃЌрЃљ рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃњрЃљрЃарЃћ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃЌ).
-   **рЃЊрЃљрЃЉрЃљрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃћ (`init_level: 2`):** рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃЊрЃарЃћ, рЃарЃљрЃЌрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃЏрЃљ (рЃЏрЃљрЃњ., `rgb_led_indicator`) рЃерЃћрЃФрЃџрЃЮрЃю рЃЏрЃўрЃАрЃў рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃљрЃюрЃЌ `start` рЃцрЃљрЃќрЃљрЃерЃў.
-   **рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЌрЃљрЃЋрЃўрЃА рЃерЃўрЃЊрЃљ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљрЃА (`is_active` рЃцрЃџрЃљрЃњрЃў), рЃарЃљрЃЌрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃўрЃфрЃўрЃџрЃЮрЃА provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃћрЃЉрЃўрЃЌрЃў рЃљрЃю рЃљрЃарЃљрЃАрЃљрЃГрЃўрЃарЃЮ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ.

## 3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ `Kconfig`-рЃўрЃА (`idf.py menuconfig`) рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА compile-time рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљрЃА.

**`Kconfig` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

`Component config` -> `Synapse Framework` -> `Modules` -> `BLE Provisioning` рЃЏрЃћрЃюрЃўрЃБрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ:
-   **`Provisioning BLE Device Name`:** рЃАрЃљрЃ«рЃћрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ BLE рЃАрЃЎрЃљрЃюрЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃА (рЃЏрЃљрЃњ., `Synapse-Prov`).
-   **`Proof-of-Possession (PoP) String`:** рЃАрЃљрЃўрЃЊрЃБрЃЏрЃџрЃЮ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА `Security 1` рЃарЃћрЃЪрЃўрЃЏрЃерЃў. рЃўрЃА рЃБрЃюрЃЊрЃљ рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЮрЃЊрЃћрЃА рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃа рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃерЃў рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџ PoP-рЃА.
-   **`Security Version`:** рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљ (0 - рЃЊрЃљрЃБрЃерЃўрЃцрЃарЃљрЃЋрЃў, 1 - рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў PoP-рЃўрЃЌ).

**`system_config.json` рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

`system_config.json`-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ `Kconfig`-рЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃцрЃљрЃарЃЋрЃљ.

```json
{
    "type": "ble_provisioning",
    "enabled": true,
    "config": {
        "instance_name": "main_ble_prov",
        "device_name": "MyCustomDevice-123",
        "pop": "secret123",
        "security_version": 1
    }
}
```

## 4. ­ЪЏа№ИЈ Service API (`ble_prov_api_t`)

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљрЃф рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ `fmw_service_get("main_ble_prov")` рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃЌ.

**API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:**

-   **`bool is_provisioning_active(void);`**  
    РЮЊ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃљрЃЏрЃЪрЃљрЃЏрЃљрЃЊ рЃљрЃЦрЃбрЃўрЃБрЃарЃў. рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА `true`-рЃА, рЃЌрЃБ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљрЃА.

-   **`esp_err_t start_provisioning(void);`**  
    ­Ъџђ рЃўрЃгрЃДрЃћрЃЉрЃА provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃљрЃЊ. рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮрЃљ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃњрЃАрЃБрЃарЃЌ рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ рЃљрЃю CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃЌ.

## 5. ­ЪЊб рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў (Events)

### рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў

| рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў | рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў (Payload) | рЃљрЃдрЃгрЃћрЃарЃљ |
|:---|:---|:---|
| `PROV_STARTED` | `NULL` | BLE provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЊрЃљрЃўрЃгрЃДрЃЮ рЃЊрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ рЃ«рЃўрЃџрЃЋрЃљрЃЊрЃўрЃљ. |
| `PROV_CREDENTIALS_RECEIVED` | `char*` (JSON) | рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃўрЃљ WiFi-рЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў. Payload рЃљрЃарЃўрЃА JSON рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў: `{"ssid":"...", "password":"..."}`. |
| `PROV_ENDED` | `NULL` | Provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ (рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃљрЃю рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃЌ). |

### рЃњрЃљрЃЏрЃЮрЃгрЃћрЃарЃўрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў

| рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў | рЃарЃћрЃљрЃЦрЃфрЃўрЃљ |
|:---|:---|
| `WIFI_EVENT_IP_ASSIGNED` | рЃЌрЃБ provisioning рЃљрЃЦрЃбрЃўрЃБрЃарЃўрЃљ, рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃА, рЃарЃљрЃЊрЃњрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ. |
| `PROV_START_REQUESTED` | рЃўрЃгрЃДрЃћрЃЉрЃА provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃА. |

## 6. ­ЪњА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

`rgb_led_indicator` рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ provisioning рЃљрЃЦрЃбрЃўрЃБрЃарЃў, рЃарЃљрЃЌрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃўрЃюрЃЊрЃўрЃЎрЃљрЃфрЃўрЃљ рЃљрЃљрЃюрЃЌрЃЮрЃА.

**`rgb_led_indicator.c`-рЃерЃў:**

```c
#include "service_locator.h"
#include "ble_prov_interface.h"

static esp_err_t rgb_led_indicator_start(module_t *self) {
    // ... (рЃбрЃљрЃАрЃЎрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃљ) ...

    // 1. ­Ъџд рЃЋрЃўрЃдрЃћрЃЉрЃЌ Provisioning рЃАрЃћрЃарЃЋрЃўрЃАрЃА
    service_handle_t prov_handle = fmw_service_get("main_ble_prov");
    if (prov_handle) {
        ble_prov_api_t *prov_api = (ble_prov_api_t *)prov_handle;

        // 2. РЮЊ рЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃЌ рЃАрЃбрЃљрЃбрЃБрЃАрЃА
        if (prov_api->is_provisioning_active()) {
            ESP_LOGI(TAG, "Provisioning is active. Setting LED to pulsing yellow.");
            // 3. ­ЪњЏ рЃЋрЃўрЃгрЃДрЃћрЃЉрЃЌ рЃДрЃЋрЃўрЃЌрЃћрЃџрЃў рЃцрЃћрЃарЃўрЃА рЃърЃБрЃџрЃАрЃљрЃфрЃўрЃљрЃА
            led_command_t cmd = {LED_MODE_PULSE, 255, 165, 0, 3000, false};
            send_command_to_task(self, &cmd);
        }
    }
    
    // ...
    return ESP_OK;
}
```

## 7. РЮЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў

-   **рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ `CMakeLists.txt`-рЃерЃў:** рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃЏрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `ble_provisioning`-рЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃЌрЃљрЃЋрЃўрЃА `CMakeLists.txt` рЃцрЃљрЃўрЃџрЃерЃў, `REQUIRES` рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃА `interfaces` (рЃЊрЃљ рЃљрЃарЃљ `ble_provisioning`, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃўрЃљ).
-   **рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ:** `Security 1` рЃарЃћрЃЪрЃўрЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ (рЃарЃЮрЃЏрЃћрЃџрЃўрЃф default-рЃўрЃљ) рЃЏрЃЎрЃљрЃфрЃарЃљрЃЊ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃарЃћрЃљрЃџрЃБрЃа рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃерЃў. рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ `PoP` рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЊрЃљ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃўрЃљ.
