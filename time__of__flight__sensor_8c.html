Detailed Description

Driver for I2C-based Time-of-Flight (TOF) distance sensor VL6180X.

Author
Synapse Framework Team
Version
1.1.0
Date
2025-07-31

This module periodically reads distance data from a VL6180X sensor via I2C and publishes the result in millimeters to the Event Bus.
