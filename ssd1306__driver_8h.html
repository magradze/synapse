Detailed Description

Public header for the SSD1306 Driver module.

This file only contains the declaration for the module's create function, which is required by the Module Factory. The actual service API provided by this module is defined in the abstract display_driver_interface.h.

module_t * ssd1306_driver_create ( const cJSON *  config)

Creates a new instance of the SSD1306 driver module.

This function is the entry point for the Module Factory. It allocates memory for the module, parses its configuration, and sets up its base lifecycle function pointers.

Parameters
[in]configA cJSON object containing the configuration for this instance. The function takes ownership of this object.
Returns
A pointer to the newly created module_t instance, or NULL on failure.
