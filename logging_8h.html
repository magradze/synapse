ეს ფაილი განსაზღვრავს ლოგირების დამხმარე მაკროს, რომელიც უზრუნველყოფს ერთიან სტილს კომპონენტის ლოგირების ტეგის (TAG) განსაზღვრისთვის. ფრეიმვორქი იყენებს სტანდარტულ esp_log.h ბიბლიოთეკას ლოგირებისთვის.

Note
ყველა .c ფაილმა, რომელსაც სჭირდება ლოგირება, უნდა გამოიყენოს DEFINE_COMPONENT_TAG მაკრო საკუთარი, უნიკალური ტეგის შესაქმნელად.

ეს მაკრო ქმნის static const char* TAG ცვლადს, რომელიც გამოიყენება ESP_LOGx მაკროებში. მისი გამოყენება უზრუნველყოფს, რომ თითოეული კომპონენტიდან წამოსული ლოგი ადვილად იდენტიფიცირებადი იყოს.

გამოყენების მაგალითი .c ფაილში:

#include "logging.h"
DEFINE_COMPONENT_TAG("MY_MODULE");
void my_function() {
ESP_LOGI(TAG, "Hello from my module!");
}
