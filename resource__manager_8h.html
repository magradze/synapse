Detailed Description

რესურსების (GPIO და სხვა) მართვის API

Version
1.0
Date
2025-06-24
Author
Giorgi Magradze

ეს კომპონენტი უზრუნველყოფს სისტემაში საერთო რესურსების (მაგ: GPIO პინების) უსაფრთხო და ექსკლუზიურ გამოყენებას. თითოეულ მოდულს შეუძლია "დალოქოს" რესურსი, რათა თავიდან აიცილოს კონფლიქტი სხვა მოდულებთან. რესურსის გათავისუფლება აუცილებელია მოდულის გაჩერების ან გამორთვისას. ეს ხელს უწყობს სისტემის სტაბილურობას და მოდულებს შორის იზოლაციას.

სანამ მოდული გამოიყენებს GPIO პინს, მან უნდა "დალოქოს" იგი, რათა სხვა მოდულმა ვერ გამოიყენოს იგივე პინი.

[in]ownerმოდულის სახელი, რომელიც იკავებს პინს (დაბგისთვის სასარგებლოა)
Returns
