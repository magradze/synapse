Detailed Description

Implementation for the SSD1306 OLED display module.

Author
Giorgi Magradze
Version
2.3.0
Date
2025-07-28

This is the final, fully functional, and corrected version. It implements a synchronous graphics API operating on an internal screen buffer, displays a splash screen, and features a dynamic header with a periodically updated WiFi status icon.

module_t * ssd1306_oled_display_create ( const cJSON *  config)

Creates a new instance of the SSD1306 OLED Display module.

This is the factory function for the module, called by the Module Factory. It allocates memory for the module and its private data, takes ownership of the provided config object, parses it, and sets up the module's base function pointers and service API.

Parameters
[in]configA pointer to the cJSON configuration object for this instance. The function takes ownership of this object.
Returns
A pointer to the newly created module_t structure, or NULL on failure.
