Detailed Description

Reports Secure Boot and Flash Encryption status.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-07-18

This module provides a centralized service to query the device's critical security features. It reads the Secure Boot and Flash Encryption status once during initialization, caches the values, and exposes them through a service API. It also provides a CLI command to display the status on the console.
