# ­ЪЊ║ рЃЏрЃЮрЃЊрЃБрЃџрЃў: `ssd1306_oled_display`

**­ЪЌѓ№ИЈ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ:** `displays`  
**Рюњ№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃў:** Giorgi Magradze & Synapse AI Expert

## 1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

`ssd1306_oled_display` рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА **рЃАрЃарЃБрЃџрЃцрЃљрЃАрЃЮрЃЋрЃљрЃюрЃў рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў**. рЃўрЃА рЃАрЃфрЃўрЃџрЃЊрЃћрЃЉрЃљ рЃБрЃЉрЃарЃљрЃџрЃЮ рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃА рЃЊрЃљ рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА **рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃБрЃљрЃџрЃБрЃа рЃАрЃљрЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃЮ рЃЊрЃљрЃцрЃљрЃА**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА SSD1306-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃЮрЃЦрЃарЃЮрЃЏрЃБрЃџрЃў OLED рЃћрЃЎрЃарЃљрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА.

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃў, рЃЉрЃБрЃцрЃћрЃарЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃюрЃЊрЃћрЃарЃўрЃюрЃњрЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃЌ. рЃДрЃЋрЃћрЃџрЃљ рЃ«рЃљрЃбрЃЋрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃ»рЃћрЃа рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃерЃўрЃЊрЃљ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЉрЃБрЃцрЃћрЃарЃерЃў рЃЊрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ `update_screen()` рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА рЃцрЃўрЃќрЃўрЃЎрЃБрЃа рЃћрЃЎрЃарЃљрЃюрЃќрЃћ. рЃћрЃА рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃњрЃџрЃБрЃЋ, "рЃљрЃфрЃўрЃЏрЃфрЃўрЃЏрЃћрЃЉрЃўрЃА" рЃњрЃљрЃарЃћрЃерЃћ рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃарЃЌрЃБрЃџрЃў рЃАрЃфрЃћрЃюрЃћрЃЉрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃ«рЃљрЃбрЃЋрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА.

## 2. ­ЪЈЏ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

- **­Ъќ╝№ИЈ рЃЉрЃБрЃцрЃћрЃарЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃюрЃЊрЃћрЃарЃўрЃюрЃњрЃў:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃдрЃљрЃа рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџ рЃбрЃљрЃАрЃЎрЃА. рЃДрЃЋрЃћрЃџрЃљ рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃў API рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ (`draw_pixel`, `draw_rect` рЃЊрЃљ рЃљ.рЃе.) рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃљрЃЊ рЃЊрЃљ рЃфрЃЋрЃџрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃерЃўрЃЊрЃљ `screen_buffer`-рЃерЃў. рЃћрЃА рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃарЃБрЃџ рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃА рЃўрЃЏрЃљрЃќрЃћ, рЃЌрЃБ рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃА рЃћрЃЎрЃарЃљрЃюрЃў.
- **­ЪцЮ рЃ░рЃўрЃЉрЃарЃўрЃЊрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃљрЃарЃўрЃА **Service Provider** рЃЊрЃљ **Event Consumer**.
  - **рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃљ:** рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃЏрЃФрЃџрЃљрЃЋрЃа `ssd1306_api_t` рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџ рЃАрЃћрЃарЃЋрЃўрЃАрЃА `Service Locator`-рЃерЃў.
  - **рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ:** рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА (`FMW_EVENT_WIFI_STATUS_READY`), рЃарЃљрЃЌрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃЮрЃА рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃЉрЃљрЃарЃў.
- **­Ъћё рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ-рЃърЃљрЃАрЃБрЃ«рЃўрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў:** WiFi рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃАрЃљрЃўрЃЏрЃћрЃЊрЃЮ рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `System Timer Service`-рЃА, рЃарЃљрЃЌрЃљ рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃљрЃЊ рЃњрЃљрЃљрЃњрЃќрЃљрЃЋрЃюрЃЮрЃА `"wifi status --silent"` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃАрЃљрЃърЃљрЃАрЃБрЃ«рЃЮрЃЊ рЃЏрЃЮрЃАрЃБрЃџ `FMW_EVENT_WIFI_STATUS_READY` рЃўрЃЋрЃћрЃюрЃЌрЃА.
- **­ЪЊЅ рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃљ `i2c_bus` рЃАрЃћрЃарЃЋрЃўрЃАрЃА I2C рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **­Ъџђ рЃЏрЃљрЃдрЃљрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃћ (`init_level: 70`):** рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃњрЃЋрЃўрЃљрЃю, рЃарЃљрЃЌрЃљ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЋрЃўрЃДрЃЮрЃЌ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА, рЃБрЃЎрЃЋрЃћ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ.

### **3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ (`config.json` рЃЊрЃљ `Kconfig`)**

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ `config.json`-рЃўрЃЌ, рЃ«рЃЮрЃџрЃЮ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ `Kconfig`-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

**`config.json` рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

  ```json
  [
    {
      "type": "ssd1306_oled_display",
      "enabled": true,
      "config": {
        "instance_name": "main_display",
        "i2c_bus_instance": "main_i2c_bus",
        "i2c_addr": "0x3C",
        "width": 128,
        "height": 64,
        "init_text": "Synapse"
      }
    }
  ]
  ```

**`config.json` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

| рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў | рЃбрЃўрЃърЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ |
| :--- | :--- | :--- | :---: |
| `instance_name` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў. рЃљрЃЏ рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ рЃ«рЃЊрЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ. | РюЁ |
| `i2c_bus_instance` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃўрЃЏ `i2c_bus` рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА `instance_name`, рЃарЃЮрЃЏрЃћрЃџрЃќрЃћрЃф рЃћрЃЎрЃарЃљрЃюрЃўрЃљ рЃЏрЃўрЃћрЃарЃЌрЃћрЃЉрЃБрЃџрЃў. | РюЁ |
| `i2c_addr` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА 7-рЃЉрЃўрЃбрЃўрЃљрЃюрЃў I2C рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў (рЃ░рЃћрЃЦрЃАрЃљрЃЊрЃћрЃфрЃўрЃЏрЃљрЃџрЃБрЃа рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў, рЃЏрЃљрЃњ., `"0x3C"`). | РюЁ |
| `width` | рЃарЃўрЃфрЃ«рЃЋрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃћ рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў. | РЮї (Default: 128) |
| `height` | рЃарЃўрЃфрЃ«рЃЋрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў. | РЮї (Default: 64) |
| `init_text` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃбрЃћрЃЦрЃАрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃљрЃА (Splash Screen). | РЮї (Default: "Synapse") |

**`Kconfig` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**
`idf.py menuconfig` -> `Component config` -> `Synapse Framework` -> `Modules` -> `SSD1306 OLED Display` рЃЏрЃћрЃюрЃўрЃБрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ:

- **WiFi Status Update Interval:** рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃА рЃърЃћрЃарЃўрЃЮрЃЊрЃў рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃћрЃЉрЃерЃў.
- **Line Text Buffer Size:** рЃерЃўрЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃЉрЃБрЃцрЃћрЃарЃўрЃА рЃќрЃЮрЃЏрЃљ.

## 4. ­Ъћї Service API (`ssd1306_api_t`)

рЃћрЃА рЃљрЃарЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃФрЃљрЃџрЃљ. рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃбрЃўрЃърЃў: **`FMW_SERVICE_TYPE_DISPLAY_API`**.

**API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:**

| рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ | рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
| :--- | :--- | :--- |
| `enable` | `(context)` | ­ЪЪб рЃарЃЌрЃљрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃА. |
| `disable` | `(context)` | РџФ рЃЌрЃўрЃерЃљрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃА. |
| `clear` | `(context)` | Рюе рЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃА **рЃерЃўрЃЊрЃљ рЃЉрЃБрЃцрЃћрЃарЃА**. |
| `update_screen` | `(context)` | ­ЪЊ▓ рЃњрЃљрЃЊрЃљрЃљрЃЦрЃЋрЃА рЃЉрЃБрЃцрЃћрЃарЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў рЃцрЃўрЃќрЃўрЃЎрЃБрЃа рЃћрЃЎрЃарЃљрЃюрЃќрЃћ. **рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃљрЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ, рЃарЃЮрЃЏ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА!** |
| `draw_pixel` | `(context, x, y, color)` | РюЈ№ИЈ рЃ«рЃљрЃбрЃљрЃЋрЃА рЃћрЃарЃЌ рЃърЃўрЃЦрЃАрЃћрЃџрЃА рЃЉрЃБрЃцрЃћрЃарЃерЃў. |
| `draw_hline` | `(context, x, y, width)` | Ръќ рЃ«рЃљрЃбрЃљрЃЋрЃА рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃа рЃ«рЃљрЃќрЃА. |
| `draw_vline` | `(context, x, y, height)` | РћЃ рЃ«рЃљрЃбрЃљрЃЋрЃА рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃа рЃ«рЃљрЃќрЃА. |
| `draw_rect` | `(context, x, y, w, h)` | ­Ъќ╝№ИЈ рЃ«рЃљрЃбрЃљрЃЋрЃА рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃЎрЃЮрЃюрЃбрЃБрЃарЃА. |
| `fill_rect` | `(context, x, y, w, h)` | РгЏ рЃ«рЃљрЃбрЃљрЃЋрЃА рЃерЃћрЃЋрЃАрЃћрЃЉрЃБрЃџ рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃА. |
| `write_text` | `(context, x, y, format, ...)` | РюЇ№ИЈ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃБрЃџ рЃбрЃћрЃЦрЃАрЃбрЃА (рЃърЃљрЃбрЃљрЃарЃљ, 6x8 рЃцрЃЮрЃюрЃбрЃўрЃЌ) рЃЉрЃБрЃцрЃћрЃарЃерЃў. |

## 5. ­Ъџђ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ

1. **рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ (Splash Screen):**
    - рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА, рЃћрЃЎрЃарЃљрЃюрЃќрЃћ 3 рЃгрЃљрЃЏрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃЊрЃўрЃЊрЃў, рЃфрЃћрЃюрЃбрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў `init_text`.
2. **рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃарЃћрЃЪрЃўрЃЏрЃў:**
    - 3 рЃгрЃљрЃЏрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃћрЃЎрЃарЃљрЃюрЃў рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃљ рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў.
    - рЃќрЃћрЃЊрЃљ рЃќрЃЮрЃџрЃерЃў (Header) рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ "synapse" рЃџрЃЮрЃњрЃЮ рЃЊрЃљ WiFi рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃўрЃЎрЃЮрЃюрЃЎрЃљ.
    - WiFi рЃўрЃЎрЃЮрЃюрЃЎрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ рЃДрЃЮрЃЋрЃћрЃџ 5 рЃгрЃљрЃЏрЃерЃў (рЃљрЃю рЃарЃЮрЃњрЃЮрЃарЃф рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ `Kconfig`-рЃерЃў).
3. **рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ:**
    - `FMW_EVENT_WIFI_STATUS_READY` рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃўрЃАрЃљрЃА, рЃ«рЃЊрЃћрЃЉрЃљ WiFi рЃўрЃЎрЃЮрЃюрЃЎрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ.
    - рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃерЃў, `FMW_EVENT_AGGREGATED_SENSOR_REPORT` рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃўрЃАрЃљрЃА, рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃљрЃарЃћрЃерЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў.

## 6. ­ЪњА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў (`selftest_manager`-рЃерЃў)

рЃћрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃЏрЃљ рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃў API.

```c
#include "service_locator.h"
#include "ssd1306_interface.h"

static void run_display_check(self_test_private_data_t *private_data) {
    // 1. рЃЋрЃўрЃдрЃћрЃЉрЃЌ рЃЊрЃўрЃАрЃърЃџрЃћрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА
    ssd1306_handle_t *display = (ssd1306_handle_t *)fmw_service_get("main_display");
    if (!display) {
        // ... рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ ...
        return;
    }

    // 2. рЃЋрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃЌ рЃерЃўрЃЊрЃљ рЃЉрЃБрЃцрЃћрЃарЃА
    display->api->clear(display->context);

    // 3. рЃЋрЃ«рЃљрЃбрЃљрЃЋрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃЉрЃБрЃцрЃћрЃарЃерЃў
    display->api->write_text(display->context, 0, 1, "Display GFX Test:");
    display->api->draw_hline(display->context, 0, 18, 128);
    display->api->draw_rect(display->context, 5, 22, 30, 20);
    display->api->fill_rect(display->context, 40, 22, 30, 20);
    display->api->write_text(display->context, 0, 6, "GFX primitives OK!");

    // 4. рЃЋрЃљрЃЕрЃћрЃюрЃЌ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ
    display->api->update_screen(display->context);

    // ... рЃЊрЃљрЃДрЃЮрЃЋрЃюрЃћрЃЉрЃљ ...

    // 5. рЃЋрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃЌ рЃћрЃЎрЃарЃљрЃюрЃА рЃбрЃћрЃАрЃбрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ
    display->api->clear(display->context);
    // рЃљрЃЦ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃЊрЃљрЃЋрЃ«рЃљрЃбрЃЮрЃЌ header-рЃў, рЃАрЃљрЃюрЃљрЃЏ рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃарЃћрЃЪрЃўрЃЏрЃА рЃЊрЃљрЃЋрЃБрЃЉрЃарЃБрЃюрЃЊрЃћрЃЉрЃўрЃЌ
    display->api->update_screen(display->context);
}
```

## 7. ­Ъџђ рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃЮ рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃА рЃўрЃЊрЃћрЃћрЃЉрЃў

- **рЃЏрЃћрЃюрЃўрЃБрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ:** рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃарЃБрЃџрЃцрЃљрЃАрЃЮрЃЋрЃљрЃюрЃў, рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЏрЃћрЃюрЃўрЃБрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.
- **рЃЏрЃарЃљрЃЋрЃљрЃџрЃў "рЃњрЃЋрЃћрЃарЃЊрЃў":** рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃўрЃЌ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ "рЃњрЃЋрЃћрЃарЃЊрЃА" (рЃЏрЃљрЃњ., рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃў, рЃљрЃЦрЃбрЃБрЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ, рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ) рЃерЃЮрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃарЃЌрЃЋрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљ.
