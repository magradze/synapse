Detailed Description

Implementation of the asynchronous Promise Manager.

Author
Synapse Framework Team
Version
2.0.0
Date
2025-08-30

This component manages the lifecycle of promises. It uses a dedicated FreeRTOS task to execute callbacks, ensuring they run in a safe, isolated context. The new architecture requires callbacks to be provided at promise creation time, eliminating race conditions.
