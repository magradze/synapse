­ЪњЙ рЃЏрЃЮрЃЊрЃБрЃџрЃў <tt>storage_manager</tt>

1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў: storage_manager
рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ: storage
рЃЋрЃћрЃарЃАрЃўрЃљ: 2.0.0
рЃљрЃЋрЃбрЃЮрЃарЃў: Synapse Framework Team

storage_manager рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃљрЃџрЃБрЃарЃў, рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃў. рЃўрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃћрЃарЃЌрЃўрЃљрЃю, рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃбрЃБрЃџ API-рЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃўрЃќрЃўрЃЎрЃБрЃа рЃАрЃљрЃфрЃљрЃЋрЃЌрЃљрЃю (backend) рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃЊ, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ:

  • NVS (Non-Volatile Storage): РџА№ИЈ рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ рЃЏрЃфрЃўрЃарЃћ рЃќрЃЮрЃЏрЃўрЃА, рЃ«рЃерЃўрЃарЃљрЃЊ рЃфрЃЋрЃџрЃљрЃЊрЃў key-value рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃЏрЃљрЃњ., рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃћрЃЉрЃў, рЃЏрЃарЃўрЃфрЃ«рЃЋрЃћрЃџрЃћрЃЉрЃў).
  • SPIFFS: ­ЪЌѓ№ИЈ рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃерЃўрЃЊрЃљ flash рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃќрЃћ, рЃерЃћрЃАрЃљрЃцрЃћрЃарЃўрЃАрЃўрЃљ рЃЋрЃћрЃЉ-рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА, рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃљрЃю рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃќрЃЮрЃЏрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ.
  • SD Card: ­ЪЌЃ№ИЈ рЃЊрЃўрЃЊрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃА рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃљрЃфрЃљрЃЋрЃў рЃњрЃљрЃарЃћ SD рЃЉрЃљрЃарЃљрЃЌрЃќрЃћ, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃўрЃА, рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃўрЃА рЃЊрЃўрЃЊрЃў рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃА рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃЌрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃАрЃарЃБрЃџрЃљрЃЊ рЃЏрЃЮрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃА рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃўрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃњрЃўрЃарЃћрЃЉрЃљ, рЃарЃљрЃЌрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃЏрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю рЃћрЃарЃЌрЃўрЃљрЃюрЃў, рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃљрЃа "рЃцрЃўрЃЦрЃарЃЮрЃЉрЃЊрЃюрЃћрЃю" рЃўрЃЏрЃљрЃќрЃћ, рЃЌрЃБ рЃАрЃљрЃЊ рЃЊрЃљ рЃарЃЮрЃњрЃЮрЃа рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃљрЃЊ.

2. ­ЪЈЌ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

  • Strategy Pattern: рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Strategy рЃЊрЃўрЃќрЃљрЃўрЃю-рЃърЃљрЃбрЃћрЃарЃюрЃА. Kconfig-рЃерЃў рЃљрЃарЃЕрЃћрЃБрЃџрЃў backend-рЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ, рЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃљрЃЊ рЃўрЃарЃЕрЃћрЃЋрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА "рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃљрЃА" (рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА) рЃЊрЃљ рЃЏрЃўрЃА API-рЃА рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА Service Locator-рЃерЃў.
  • рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў (Service-driven): рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА рЃАрЃБрЃцрЃЌрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃџрЃў. рЃўрЃА рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА storage_api_t рЃАрЃћрЃарЃЋрЃўрЃАрЃА рЃЊрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃќрЃћ.
  • рЃўрЃќрЃЮрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ Namespace:
    • NVS Backend: рЃўрЃДрЃћрЃюрЃћрЃЉрЃА "рЃЏрЃцрЃџрЃЮрЃЉрЃћрЃџрЃўрЃА" (owner) рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃА, рЃарЃЮрЃњрЃЮрЃарЃф NVS-рЃўрЃА namespace-рЃА, рЃарЃљрЃф рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃљрЃЋрЃА рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃА.
    • SPIFFS/SD Card Backends: рЃўрЃДрЃћрЃюрЃћрЃЉрЃА owner-рЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃА, рЃарЃљрЃЌрЃљ рЃЏрЃЮрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЊрЃљрЃ»рЃњрЃБрЃцрЃћрЃЉрЃљ рЃЊрЃљ рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃљ.
  • **рЃЊрЃљрЃЉрЃљрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃћ (init_level: 20):** рЃарЃЮрЃњрЃЮрЃарЃф рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃЊрЃарЃћрЃБрЃџ рЃћрЃбрЃљрЃърЃќрЃћ.
  • рЃћрЃарЃЌрЃўрЃљрЃюрЃў API: рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃљрЃарЃЕрЃћрЃБрЃџрЃў backend-рЃўрЃАрЃљ, рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю рЃћрЃарЃЌрЃАрЃљ рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ storage_api_t рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА рЃарЃЮрЃњрЃЮрЃарЃф key-value, рЃўрЃАрЃћ рЃцрЃљрЃўрЃџрЃБрЃа рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃА.

3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ Kconfig-рЃўрЃА (idf.py menuconfig) рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ, рЃАрЃљрЃЊрЃљрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃљрЃўрЃарЃЕрЃўрЃЮрЃЌ backend-рЃў рЃЊрЃљ рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ рЃЏрЃўрЃАрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў.

**system_config.json рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

{
"type": "storage_manager",
"enabled": true,
"config": {
"instance_name": "main_storage"
}
}

**Kconfig рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

Component config -> Synapse Framework -> Modules -> Storage Manager рЃЏрЃћрЃюрЃўрЃБрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ:

  1. рЃљрЃўрЃарЃЕрЃўрЃЮрЃЌ Storage Backend: NVS, SPIFFS, рЃљрЃю SD Card.
  2. рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ SPIFFS: рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃЮрЃЌ рЃърЃљрЃарЃбрЃўрЃфрЃўрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў, рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃЊрЃљ рЃљ.рЃе.
  3. рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ SD Card: рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃЮрЃЌ mount point-рЃў рЃЊрЃљ SPI рЃърЃўрЃюрЃћрЃЉрЃўрЃА рЃюрЃЮрЃЏрЃарЃћрЃЉрЃў (MISO, MOSI, CLK, CS).

4. ­ЪЏа№ИЈ Service API (<tt>storage_api_t</tt>)

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА рЃбрЃўрЃърЃўрЃЌ FMW_SERVICE_TYPE_NVRAM_API. рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃљрЃЏ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ fmw_service_get("main_storage") рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃЌ.
