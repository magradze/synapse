module_t * spi_bus_create ( const cJSON *  config)

Creates a new instance of the SPI Bus module.

This function is the entry point for creating an SPI Bus module. It allocates memory, parses the provided JSON configuration, and sets up the module's lifecycle function pointers.

Parameters
[in]configA pointer to the cJSON object containing the module's configuration. The module takes ownership of this object and will free it during deinitialization.
Returns
A pointer to the newly created module_t instance, or NULL on failure.
