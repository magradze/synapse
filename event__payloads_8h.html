Detailed Description

ტელემეტრიისა და სისტემური ივენთების მონაცემთა სტრუქტურები.

ეს ფაილი განსაზღვრავს მონაცემთა სტრუქტურებს (payloads), რომლებიც გამოიყენება Synapse ESP Framework-ის Event Bus-ში. ის ასევე შეიცავს დამხმარე ფუნქციებს ამ სტრუქტურების მეხსიერების უსაფრთხო მართვისთვის.

ათავისუფლებს ტელემეტრიის პეილოდს.
