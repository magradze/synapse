Detailed Description

Public API for consuming asynchronous operations using a Promise pattern.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-08-29

This header defines the consumer-facing API for interacting with promises. Modules that initiate an asynchronous operation will receive a promise handle and can use the functions defined here to register callbacks for success or failure scenarios.

Registers a callback to be executed upon failure of the promise.

If the promise has already been rejected, the callback may be executed immediately.

Parameters
[in]handleThe promise handle.
[in]catch_cbThe function to call when the promise is rejected.
[in]user_contextA context pointer to be passed to the callback. Can be NULL.
Returns
  • ESP_OK: If the callback was successfully registered.
  • ESP_ERR_INVALID_ARG: If the handle or callback is NULL.
  • ESP_ERR_INVALID_STATE: If a 'catch' callback is already registered for this promise.

◆ fmw_promise_destroy()

void fmw_promise_destroy ( promise_handle_t  handle)

Destroys a promise handle and releases its associated resources.

This function should be called by the consumer if it is no longer interested in the result of the promise. It signals to the promise manager that the resources can be cleaned up. The manager will also automatically clean up promises after they are executed.

Parameters
[in]handleThe promise handle to destroy. Can be NULL.

Registers a callback to be executed upon successful completion of the promise.

If the promise has already been resolved, the callback may be executed immediately from the context of the promise manager's task.

Parameters
[in]handleThe promise handle returned by an asynchronous API call.
[in]then_cbThe function to call when the promise is resolved.
[in]user_contextA context pointer to be passed to the callback. Can be NULL.
Returns
  • ESP_OK: If the callback was successfully registered.
  • ESP_ERR_INVALID_ARG: If the handle or callback is NULL.
  • ESP_ERR_INVALID_STATE: If a 'then' callback is already registered for this promise.
