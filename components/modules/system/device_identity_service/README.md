# ­Ъєћ рЃЏрЃЮрЃЊрЃБрЃџрЃў: `device_identity_service`

## 1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

**рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў:** `device_identity_service`  
**­ЪЌѓ№ИЈ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ:** `system`  
**Рюњ№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃў:** Synapse Framework Team  
**­Ъћќ рЃЋрЃћрЃарЃАрЃўрЃљ:** 1.2.0

`Device Identity Service` рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА **рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў, рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃЏрЃЮрЃЊрЃБрЃџрЃў**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџ рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋ рЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃАрЃљ рЃЊрЃљ firmware-рЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃќрЃћ.

рЃЏрЃўрЃАрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃљ, рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА рЃћрЃарЃЌрЃ«рЃћрЃџ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў (MAC рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў) рЃЊрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ, рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃА рЃўрЃАрЃўрЃюрЃў рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ, `Service Locator`-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ, рЃЏрЃўрЃљрЃгрЃЮрЃЊрЃЮрЃА рЃћрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃА. рЃћрЃА рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃљрЃЋрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃљрЃА рЃЊрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃЊрЃћрЃА рЃћрЃарЃЌрЃАрЃљ рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ, рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃБрЃџ рЃАрЃљрЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА.

## 2. ­ЪЈЏ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

- **рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў (Service-driven):** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА рЃАрЃБрЃцрЃЌрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃџрЃў. рЃўрЃА рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА `device_identity_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА `Service Locator`-рЃерЃў рЃЊрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃќрЃћ.
- **рЃљрЃЊрЃарЃћрЃБрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ (`init_level: 20`):** рЃарЃЮрЃњрЃЮрЃарЃф рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃЊрЃарЃћрЃБрЃџ рЃћрЃбрЃљрЃърЃќрЃћ, рЃарЃљрЃЌрЃљ рЃЏрЃўрЃАрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃўрЃДрЃЮрЃА рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЎрЃћрЃерЃўрЃарЃћрЃЉрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃ«рЃћрЃџ, `init` рЃцрЃљрЃќрЃљрЃерЃў рЃЊрЃљ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃљрЃЌ рЃЌрЃљрЃЋрЃўрЃА `private_data` рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃерЃў. рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃў API рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃћрЃЉрЃў рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃљрЃЊ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃћрЃю рЃљрЃЏ рЃЎрЃћрЃерЃўрЃарЃћрЃЉрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА.
- **CLI рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃЊрЃљ рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃљ:** `device info` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃАрЃљрЃА, рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃерЃў, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА `SYNAPSE_EVENT_DEVICE_INFO_READY` рЃўрЃЋрЃћрЃюрЃЌрЃА, рЃарЃљрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА `mqtt_manager`-рЃА, рЃћрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ MQTT-рЃўрЃЌ рЃњрЃљрЃљрЃЋрЃарЃфрЃћрЃџрЃЮрЃА.

## 3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃљрЃЏ рЃЏрЃЮрЃЊрЃБрЃџрЃА рЃљрЃа рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ рЃарЃЌрЃБрЃџрЃў, рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў. рЃћрЃарЃЌрЃљрЃЊрЃћрЃарЃЌрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃљрЃарЃўрЃА `instance_name`.

**`system_config.json`-рЃерЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

```json
{
    "type": "device_identity_service",
    "enabled": true,
    "config": {
        "instance_name": "main_identity_service"
    }
}
```

## 4. ­Ъћї Service API (`device_identity_api_t`)

рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃю рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃљрЃЏ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ `Service Locator`-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ. рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃбрЃўрЃърЃў: **`SYNAPSE_SERVICE_TYPE_DEVICE_IDENTITY_API`**.

**API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:**

- **`const char* get_device_id(void);`**  
    ­Ъєћ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа ID-рЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ MAC рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃАрЃњрЃљрЃю (рЃЏрЃљрЃњ., `"5C013B7246E4"`).
- **`const char* get_firmware_version(void);`**  
    ­Ъћќ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА firmware-рЃўрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЋрЃћрЃарЃАрЃўрЃљрЃА (рЃЏрЃљрЃњ., `"v3.2.1"`).

## 5. ­ЪЊб рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў (Events)

### рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў

| рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў | рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў (Payload) | рЃљрЃдрЃгрЃћрЃарЃљ |
| :--- | :--- | :--- |
| `SYNAPSE_EVENT_DEVICE_INFO_READY` | `char*` (JSON) | ­ЪЊі `device info` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃърЃљрЃАрЃБрЃ«рЃў JSON рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў. |

### рЃњрЃљрЃЏрЃЮрЃгрЃћрЃарЃўрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃў

| рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў | рЃарЃћрЃљрЃЦрЃфрЃўрЃљ |
| :--- | :--- |
| `SYNAPSE_EVENT_SYSTEM_START_COMPLETE` | рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃЌрЃљрЃЋрЃўрЃА CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃА, рЃарЃљрЃЊрЃњрЃљрЃю рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃАрЃарЃБрЃџрЃљрЃЊ рЃњрЃљрЃћрЃерЃЋрЃљ. |

## 6. ­ЪЊА MQTT рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў

- **рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃћрЃџрЃў рЃЌрЃћрЃЏрЃљ:** `synapse/devices/{id}/state/device/info`
- **рЃарЃЮрЃЊрЃўрЃА рЃЦрЃЋрЃћрЃДрЃюрЃЊрЃћрЃЉрЃљ:** рЃарЃЮрЃЊрЃћрЃАрЃљрЃф `mqtt_manager` рЃЏрЃўрЃўрЃдрЃћрЃЉрЃА `SYNAPSE_EVENT_DEVICE_INFO_READY` рЃўрЃЋрЃћрЃюрЃЌрЃА.
- **Payload (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў):**


## 8. ­ЪњА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў (рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ-рЃърЃљрЃАрЃБрЃ«рЃў)

1. **­Ъњ╗ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў (Postman/MQTT рЃЎрЃџрЃўрЃћрЃюрЃбрЃў):**
    - **рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА:** `synapse/devices/+/state/device/info`
    - **рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА:**
        - **Topic:** `synapse/devices/5C013B7246E4/cmd/in`
        - **Payload:** `"device info"`

2. **РџЎ№ИЈ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ (Synapse Framework):**
    - `mqtt_manager` рЃўрЃдрЃћрЃЉрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃўрЃЋрЃћрЃюрЃЌрЃА `Command Router`-рЃерЃў.
    - `command_router` рЃўрЃФрЃљрЃ«рЃћрЃЉрЃА `device_identity_service`-рЃўрЃА `cmd_handler`-рЃА.
    - `device_identity_service` рЃљрЃњрЃарЃЮрЃЋрЃћрЃЉрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА, рЃЦрЃЏрЃюрЃўрЃА JSON-рЃА рЃЊрЃљ рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА `SYNAPSE_EVENT_DEVICE_INFO_READY` рЃўрЃЋрЃћрЃюрЃЌрЃА.
    - `mqtt_manager` рЃўрЃдрЃћрЃЉрЃА `SYNAPSE_EVENT_DEVICE_INFO_READY` рЃўрЃЋрЃћрЃюрЃЌрЃА рЃЊрЃљ рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА MQTT-рЃерЃў.

3. **­Ъњ╗ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў (Postman/MQTT рЃЎрЃџрЃўрЃћрЃюрЃбрЃў):**
    - **рЃўрЃдрЃћрЃЉрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљрЃА:**
        - **Topic:** `synapse/devices/5C013B7246E4/state/device/info`
        - **Payload:** `{"device_id":"5C013B7246E4", "firmware_version":"v3.2.1"}`

## 9. РЮЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў

- **рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ `CMakeLists.txt`-рЃерЃў:** `device_identity_service`-рЃўрЃА `CMakeLists.txt` рЃцрЃљрЃўрЃџрЃерЃў, `REQUIRES` рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў `interfaces`, рЃарЃљрЃЌрЃљ рЃЏрЃљрЃА рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљ `Command Router`-рЃўрЃА API рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃќрЃћ.
- **рЃЊрЃљрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ:** CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ `SYNAPSE_EVENT_SYSTEM_START_COMPLETE` рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ `Command Router` рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃљрЃЏ рЃЊрЃарЃЮрЃА рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃарЃљрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃљрЃЊ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ.
