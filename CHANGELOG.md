# CHANGELOG

## [9.0.0] - 2025-09-01

### ­ЪњЦ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў (Breaking Changes & Refactoring)

- **refactor(core):** ­ЪЈи№ИЈ рЃњрЃљрЃюрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЊрЃљ рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА API-рЃА рЃАрЃарЃБрЃџрЃў рЃарЃћрЃЉрЃарЃћрЃюрЃЊрЃўрЃюрЃњрЃў.
  - **рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃћрЃЉрЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ:** рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃ»рЃљрЃарЃЮ API рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА, рЃбрЃўрЃърЃўрЃА рЃЊрЃљ рЃЏрЃљрЃЎрЃарЃЮрЃА рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃў рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ `fmw_`-рЃЊрЃљрЃю `synapse_`-рЃќрЃћ (рЃЊрЃљ `FMW_`-рЃЊрЃљрЃю `SYNAPSE_`-рЃќрЃћ). рЃћрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃА рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃЮрЃЉрЃљрЃА рЃЊрЃљ рЃЎрЃЮрЃЊрЃА рЃБрЃцрЃарЃЮ рЃфрЃ«рЃљрЃЊрЃА рЃ«рЃЊрЃўрЃА.
  - **рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ:** рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃљ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃЏрЃЮрЃћрЃДрЃћрЃюрЃћрЃЉрЃўрЃюрЃљрЃЌ рЃљрЃ«рЃљрЃџрЃў, `synapse_` рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ API.

### ­ЪЊё рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ (Documentation)

- **docs(project):** ­Ъџђ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃљ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў `README.md` рЃцрЃљрЃўрЃџрЃў.
  - **рЃАрЃарЃБрЃџрЃў рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃў:** рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃўрЃЌрЃљрЃарЃњрЃЏрЃюрЃљ рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃа рЃћрЃюрЃљрЃќрЃћ, рЃарЃљрЃЌрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃБрЃцрЃарЃЮ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃўрЃДрЃЮрЃА рЃАрЃљрЃћрЃарЃЌрЃљрЃерЃЮрЃарЃўрЃАрЃЮ рЃљрЃБрЃЊрЃўрЃбрЃЮрЃарЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.
  - **рЃЏрЃљрЃюрЃўрЃцрЃћрЃАрЃбрЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ:** рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ рЃЉрЃЏрЃБрЃџрЃў рЃЊрЃљ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃў рЃљрЃ«рЃљрЃџрЃў `manifesto.md` рЃцрЃљрЃўрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
