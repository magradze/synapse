esp_err_t(* timer_api_t::cancel_event) (fmw_timer_handle_t handle)

Cancels a previously scheduled event.

Parameters
[in]handleThe handle of the timer to cancel, obtained from schedule_event.
Returns
  • ESP_OK: If the timer was successfully found and cancelled.
  • ESP_ERR_INVALID_ARG: If the handle is NULL or invalid.
  • ESP_ERR_NOT_FOUND: If no timer with the given handle was found.

fmw_timer_handle_t(* timer_api_t::schedule_event) (const char *event_name, uint32_t interval_ms, bool is_periodic)

Schedules an event to be posted on the Event Bus after a specified interval.

Parameters
[in]event_nameThe name of the event to post. The string must be persistent.
[in]interval_msThe interval in milliseconds after which to post the event.
[in]is_periodicIf true, the event will be posted repeatedly at the given interval. If false, it will be a one-shot timer.
Returns
A handle to the scheduled timer, or NULL on failure (e.g., no more timer slots).
