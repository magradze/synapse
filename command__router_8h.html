module_t * command_router_create ( const cJSON *  config)

Creates a new instance of the Command Router module.

This function allocates and initializes all necessary resources for the module based on the provided JSON configuration.

Parameters
[in]configA cJSON object containing the module's configuration, typically from system_config.json. Can be NULL to use defaults from Kconfig.
Returns
A pointer to the created module (module_t*), or NULL on failure.
Note
The created module must be registered with the system_manager to be part of the system's lifecycle.
Warning
The caller is responsible for ensuring the module's memory is eventually freed by calling its base.deinit() function.
