Detailed Description

მოდულების რეგისტრაციისა და მართვის იმპლემენტაცია.

Version
2.2
Date
2025-06-27
Author
Giorgi Magradze

ეს ფაილი პასუხისმგებელია სისტემის ინიციალიზაციისას system_config.json-ის წაკითხვაზე, Module Factory-ის გამოყენებით მოდულების დინამიურ შექმნასა და მათ ცენტრალიზებულ რეესტრში რეგისტრაციაზე.

Warning
დაბრუნებული მაჩვენებლის შეცვლის უფლება აქვს მხოლოდ System Manager-ს.
