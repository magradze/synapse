Detailed Description

Universal Service API contract for display drivers.

This file defines the abstract interface that all display drivers (e.g., SSD1306, ST7789) in the Synapse Framework must implement. Its purpose is to completely decouple high-level UI logic (from ui_manager) from the low-level hardware implementation.
