◆ fmw_event_bus_init()

esp_err_t fmw_event_bus_init ( void  )

Event Bus-ის ინიციალიზაცია და ფონური ტასკის გაშვება.

ეს ფუნქცია ქმნის ივენთების რიგს (queue) და აამუშავებს ფონურ ტასკს, რომელიც პასუხისმგებელია რიგში მოხვედრილი ივენთების დამუშავებასა და შესაბამისი გამომწერების ინფორმირებაზე. უნდა გამოიძახოს system_manager-მა ინიციალიზაციისას.

Returns
esp_err_t ოპერაციის წარმატების კოდი.
Return values
ESP_OKთუ ინიციალიზაცია წარმატებით დასრულდა.
ESP_FAILთუ ტასკის ან რიგის შექმნა ვერ მოხერხდა.

esp_err_t fmw_event_bus_post ( const char *  event_name,
struct event_data_wrapper_t *  data_wrapper

აქვეყნებს ივენთს Event Bus-ზე ასინქრონულად.

ეს ფუნქცია ამატებს ივენთს და მის თანდართულ მონაცემებს (თუ არსებობს) Event Bus-ის რიგში. ფონური ტასკი შემდგომში ამ ივენთს მიაწვდის ყველა იმ მოდულს, რომელსაც გამოწერილი აქვს ეს event_name.

Parameters
[in]event_nameივენთის უნიკალური სახელი (სტრიქონი).
[in]data_wrapperმაჩვენებელი "შეფუთულ" მონაცემებზე. თუ ივენთს მონაცემები არ სჭირდება, ეს პარამეტრი უნდა იყოს NULL.
Returns
esp_err_t ოპერაციის წარმატების კოდი.
Return values
ESP_OKთუ ივენთი წარმატებით დაემატა რიგში.
ESP_ERR_NO_MEMთუ მეხსიერება ვერ გამოიყო ივენთის სახელის კოპირებისთვის.
ESP_FAILთუ რიგი სავსეა.

არეგისტრირებს მოდულს, როგორც კონკრეტული ივენთის გამომწერს.

როდესაც მითითებული event_name-ის მქონე ივენთი გამოქვეყნდება, Event Bus გამოიძახებს ამ module-ის handle_event ფუნქციას.

Parameters
[in]event_nameივენთის უნიკალური სახელი (სტრიქონი), რომელზეც გამოწერა ხდება.
[in]moduleმაჩვენებელი მოდულზე, რომელიც ამ ივენთს გამოიწერს.
Returns
esp_err_t ოპერაციის წარმატების კოდი.
Return values
ESP_OKთუ გამოწერა წარმატებით შესრულდა (ან მოდული უკვე გამოწერილი იყო).
ESP_ERR_NO_MEMთუ მეხსიერება არასაკმარისია ახალი გამომწერის დასამატებლად.
ESP_ERR_INVALID_ARGთუ module ან event_name არასწორია.
