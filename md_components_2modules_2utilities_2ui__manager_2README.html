­ЪќЦ№ИЈ рЃЏрЃЮрЃЊрЃБрЃџрЃў: <tt>ui_manager</tt>

­ЪЌѓ№ИЈ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ: utilities Рюњ№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃў: Synapse Framework Team ­Ъћќ рЃЋрЃћрЃарЃАрЃўрЃљ: 4.2.0

1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

ui_manager рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА (UI) рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃБрЃарЃў "рЃЊрЃўрЃарЃўрЃЪрЃЮрЃарЃў". рЃЏрЃўрЃАрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃљ, рЃЏрЃљрЃарЃЌрЃЮрЃА UI-рЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ, рЃЏрЃЮрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃА рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃћрЃџрЃћрЃњрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃфрЃўрЃљ рЃњрЃљрЃБрЃгрЃўрЃЮрЃА UI-рЃА рЃЊрЃљрЃюрЃљрЃгрЃћрЃЋрЃарЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃА.

рЃћрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃўрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃАрЃарЃБрЃџрЃљрЃЊ рЃўрЃДрЃЮрЃА рЃњрЃљрЃЏрЃўрЃ»рЃюрЃБрЃџрЃў (decoupled) рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃўрЃАрЃњрЃљрЃю. рЃўрЃА рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА рЃќрЃЮрЃњрЃљрЃЊ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃЌрЃљрЃю display_driver_interface-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф Service Locator-рЃўрЃЊрЃљрЃю рЃўрЃдрЃћрЃЉрЃА.

2. ­ЪЈЏ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

ui_manager рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљрЃюрЃљрЃгрЃћрЃЋрЃарЃћрЃЉрЃБрЃџ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃќрЃћ, рЃАрЃљрЃЊрЃљрЃф рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ рЃфрЃљрЃџрЃЎрЃћ рЃџрЃЮрЃњрЃўрЃЎрЃБрЃа рЃћрЃарЃЌрЃћрЃБрЃџрЃљрЃЊ:

  • **ui_manager.c (рЃЊрЃўрЃарЃўрЃЪрЃЮрЃарЃў):** ­ЪДа рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃцрЃљрЃўрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃўрЃА рЃфрЃўрЃЎрЃџрЃќрЃћ (init, start, deinit), рЃАрЃљрЃГрЃўрЃарЃЮ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃърЃЮрЃЋрЃћрЃЉрЃљрЃќрЃћ рЃЊрЃљ UI-рЃА рЃќрЃЮрЃњрЃљрЃЊрЃў рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃюрЃЦрЃљрЃюрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃќрЃћ.
  • **ui_events.c (рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃћрЃџрЃў):** ­ЪЊг рЃўрЃдрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџ рЃўрЃЋрЃћрЃюрЃЌрЃА Event Bus-рЃўрЃЊрЃљрЃю. рЃўрЃА рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃўрЃА "рЃњрЃљрЃдрЃЋрЃўрЃФрЃћрЃЉрЃўрЃА" рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА, рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃўрЃА cooldown-рЃА рЃЊрЃљ рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃбрЃўрЃърЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃўрЃФрЃљрЃ«рЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА (рЃЏрЃљрЃњ., рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА ui_menu-рЃА, рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЎрЃў рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃфрЃЋрЃџрЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљрЃА).
  • **ui_menu.c (рЃЏрЃћрЃюрЃўрЃБрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ):** ­ЪДГ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЏрЃћрЃюрЃўрЃБрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљрЃА, рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА (UP/DOWN/OK) рЃЊрЃљ рЃЏрЃћрЃюрЃўрЃБрЃА рЃћрЃЎрЃарЃљрЃюрЃћрЃЉрЃўрЃА рЃ«рЃљрЃбрЃЋрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃА.
  • **ui_screens.c (рЃћрЃЎрЃарЃљрЃюрЃћрЃЉрЃўрЃА рЃ«рЃљрЃбрЃЋрЃљ):** ­Ъје рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃљрЃарЃљрЃЏрЃћрЃюрЃўрЃБрЃА рЃћрЃЎрЃарЃљрЃюрЃћрЃЉрЃўрЃА (Splash, Home) рЃЊрЃљ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА (Header, Footer Buttons, Icons) рЃЊрЃљрЃ«рЃљрЃбрЃЋрЃљрЃќрЃћ.

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў:

  • Service Consumer: рЃћрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў.
  • Event Consumer: рЃўрЃА рЃБрЃАрЃЏрЃћрЃюрЃА FMW_EVENT_BUTTON_PRESSED рЃЊрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА.
  • State Machine: рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃерЃўрЃЊрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљрЃќрЃћ (ui_state_t).
  • рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃарЃћрЃюрЃЊрЃћрЃарЃўрЃюрЃњрЃў: рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА рЃўрЃдрЃћрЃЉрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃќрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃњрЃљрЃюрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃА рЃљрЃЏ рЃќрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.

3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЊрЃћрЃЉрЃљ рЃЏрЃўрЃАрЃў config.json рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃбрЃўрЃърЃў рЃљрЃдрЃгрЃћрЃарЃљ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ
instance_name рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў UI рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃўрЃА рЃљрЃЏ рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў. РюЁ
display_driver_service рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃўрЃЏ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА instance_name, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЏ UI рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА рЃЊрЃљрЃАрЃљрЃ«рЃљрЃбрЃљрЃЊ. РюЁ

4. ­Ъџђ UI-рЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃЊрЃљ рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ

  • Splash Screen: рЃЕрЃљрЃарЃЌрЃЋрЃўрЃАрЃљрЃА, 3 рЃгрЃљрЃЏрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА "SYNAPSE" рЃџрЃЮрЃњрЃЮрЃА.
