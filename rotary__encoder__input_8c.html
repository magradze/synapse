Detailed Description

A universal module to handle multiple rotary encoder inputs and publish standard navigation events.

Author
Synapse Framework Team
Version
2.0.0
Date
2025-08-24

This module reads multiple rotary encoder's rotation and button presses, debounces the signals, and translates them into standard SYNAPSE_EVENT_BUTTON_PRESSED events with "UP", "DOWN", and "OK" payloads. It supports both direct GPIO connection and connection via an MCP23017 I/O expander. All instances are managed by a single, shared FreeRTOS task for resource efficiency.
