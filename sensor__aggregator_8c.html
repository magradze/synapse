Detailed Description

Implementation of the Sensor Aggregator module.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-07-17

This module listens for individual sensor events, aggregates their data into a single snapshot, and publishes a consolidated report based on a configured strategy (time window or trigger event).
