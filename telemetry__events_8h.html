Detailed Description

ტელემეტრიისა და სისტემური ივენთების მონაცემთა სტრუქტურები და შესაბამისი cleanup ფუნქციები.

ეს ფაილი განსაზღვრავს სტრუქტურებსა და დამხმარე ფუნქციებს Synapse ESP Framework-ის Event Bus-ისთვის, მათ შორის: ტელემეტრიის, კონფიგურაციის განახლების, მოდულის ჩართვა/გამორთვისა და ჯანმრთელობის მონიტორინგის ივენთებისთვის.
