Detailed Description

Public API for the Shared Task Pool Manager.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-09-06

This core component provides a pool of worker tasks to execute short, periodic, or one-shot jobs on behalf of other modules. This significantly reduces system overhead by avoiding the need for each module to create its own task for simple background work.

esp_err_t synapse_task_pool_cancel_job ( synapse_job_handle_t  handle)

Cancels a previously scheduled job.

Parameters
[in]handleThe handle of the job to cancel, obtained from schedule_job.
Returns
  • ESP_OK: If the job was successfully found and cancelled.
  • ESP_ERR_INVALID_ARG: If the handle is NULL.
  • ESP_ERR_NOT_FOUND: If no job with the given handle was found.

Schedules a new job for execution by the task pool.

Parameters
[in]job_functionThe function to be executed.
[in]user_contextA context pointer to be passed to the job function.
[in]interval_msThe interval in milliseconds for periodic execution. If 0, the job is a one-shot and will run only once.
[in]is_periodicIf true, the job will be rescheduled after each execution.
Returns
A handle to the scheduled job, or NULL on failure. The handle is required to cancel the job later.
