esp_err_t(* watchdog_api_t::register_heartbeat) (const char *module_name)

Registers a module for heartbeat monitoring.

After registration, the Watchdog Manager will expect this module to periodically call send_heartbeat().

Parameters
[in]module_nameThe unique instance name of the module to monitor.
Returns
ESP_OK on success, ESP_ERR_NO_MEM if the list is full.
