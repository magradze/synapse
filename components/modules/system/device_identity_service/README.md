# ­Ъєћ рЃЏрЃЮрЃЊрЃБрЃџрЃў: Device Identity Service

**­ЪЌѓ№ИЈ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ:** `system`  
**Рюњ№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃў:** Synapse Framework Team  
**­Ъћќ рЃЋрЃћрЃарЃАрЃўрЃљ:** 1.1.0

## 1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

`Device Identity Service` рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў, рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃЏрЃЮрЃЊрЃБрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџ рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋ рЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃАрЃљ рЃЊрЃљ firmware-рЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃќрЃћ.

рЃЏрЃўрЃАрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃљ, рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА рЃћрЃарЃЌрЃ«рЃћрЃџ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў (рЃЏрЃљрЃњ., MAC рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў) рЃЊрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ, рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃА рЃўрЃАрЃўрЃюрЃў рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ, `Service Locator`-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ, рЃЏрЃўрЃљрЃгрЃЮрЃЊрЃЮрЃА рЃћрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃА, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃўрЃА рЃЊрЃљрЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ. рЃћрЃА рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃљрЃЋрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃљрЃА рЃЊрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃЊрЃћрЃА рЃћрЃарЃЌрЃАрЃљ рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ, рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃБрЃџ рЃАрЃљрЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА.

## 2. ­ЪЈЏ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

- **рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў (Service-driven):** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА рЃАрЃБрЃцрЃЌрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃџрЃў. рЃўрЃА рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА `device_identity_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА `Service Locator`-рЃерЃў рЃЊрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃќрЃћ.
- **рЃљрЃЊрЃарЃћрЃБрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ (`init_level: 20`):** рЃарЃЮрЃњрЃЮрЃарЃф рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃЊрЃарЃћрЃБрЃџ рЃћрЃбрЃљрЃърЃќрЃћ, рЃарЃљрЃЌрЃљ рЃЏрЃўрЃАрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃўрЃДрЃЮрЃА рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃљрЃЌрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃерЃў.
- **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЎрЃћрЃерЃўрЃарЃћрЃЉрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃ«рЃћрЃџ, `init` рЃцрЃљрЃќрЃљрЃерЃў рЃЊрЃљ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃљрЃЌ рЃЌрЃљрЃЋрЃўрЃА `private_data` рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃерЃў. рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃў API рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃћрЃЉрЃў рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃљрЃЊ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃћрЃю рЃљрЃЏ рЃЎрЃћрЃерЃўрЃарЃћрЃЉрЃБрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА, рЃарЃљрЃф рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃАрЃгрЃарЃљрЃцрЃА рЃЊрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃА рЃ«рЃЊрЃўрЃА.
- **CLI рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА "рЃЊрЃљрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА" рЃърЃљрЃбрЃћрЃарЃюрЃА. рЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА `FMW_EVENT_SYSTEM_START_COMPLETE` рЃўрЃЋрЃћрЃюрЃЌрЃА рЃЊрЃљ рЃЌрЃљрЃЋрЃўрЃА CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА (`device info`) рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃљрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃарЃљрЃф рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЏ `Command Router` рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃЏрЃќрЃљрЃЊ рЃљрЃарЃўрЃА.

## 3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃљрЃЏ рЃЏрЃЮрЃЊрЃБрЃџрЃА рЃљрЃа рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ рЃарЃЌрЃБрЃџрЃў, рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃА рЃЌрЃљрЃЋрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃўрЃЊрЃљрЃю рЃЊрЃљ firmware-рЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃЊрЃљрЃю рЃўрЃдрЃћрЃЉрЃА. рЃћрЃарЃЌрЃљрЃЊрЃћрЃарЃЌрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃљрЃарЃўрЃА `instance_name`.

**`system_config.json`-рЃерЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

```json
{
    "type": "device_identity_service",
    "enabled": true,
    "config": {
        "instance_name": "main_identity_service"
    }
}
```

**рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

| рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў | рЃбрЃўрЃърЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ | Default (`Kconfig`) |
| :--- | :--- | :--- | :---: | :--- |
| `instance_name` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў. | РЮї | `main_identity_service` |

## 4. ­Ъћї Service API (`device_identity_api_t`)

рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃю рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃљрЃЏ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ `Service Locator`-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ. рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃбрЃўрЃърЃў: **`FMW_SERVICE_TYPE_DEVICE_IDENTITY_API`**.

**API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:**

- **`const char* get_device_id(void);`**  
    ­Ъєћ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа ID-рЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ MAC рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃАрЃњрЃљрЃю (рЃЏрЃљрЃњ., `"5C013B7246E4"`). рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃќрЃћ рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЌрЃљрЃЋрЃљрЃЊ рЃАрЃћрЃарЃЋрЃўрЃАрЃў.

- **`const char* get_firmware_version(void);`**  
    ­Ъћќ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА firmware-рЃўрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЋрЃћрЃарЃАрЃўрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃдрЃћрЃЉрЃБрЃџрЃўрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃЊрЃљрЃю (рЃЏрЃљрЃњ., `"v3.2.0"`).

## 5. Рїе№ИЈ CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў (Command Line Interface)

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА `device` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃФрЃџрЃћрЃЋрЃљ рЃАрЃљрЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЏрЃўрЃдрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА.

| рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ | рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
| :--- | :--- | :--- |
| `device info` | - | Рё╣№ИЈ рЃњрЃљрЃЏрЃЮрЃљрЃЦрЃЋрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў ID рЃЊрЃљ firmware-рЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљ рЃАрЃћрЃарЃўрЃБрЃџ рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃерЃў. |

**рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃў:**

```bash
SYNAPSE> device info
----------------------------------
  Device Information
----------------------------------
  Device ID           : 5C013B7246E4
  Firmware Version    : v3.2.0
----------------------------------
```

## 6. ­ЪњА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

рЃгрЃљрЃарЃЏрЃЮрЃЋрЃўрЃЊрЃњрЃўрЃюрЃЮрЃЌ, рЃарЃЮрЃЏ `mqtt_module`-рЃА рЃАрЃБрЃарЃА, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў `ClientID` рЃЉрЃарЃЮрЃЎрЃћрЃарЃЌрЃљрЃю рЃЊрЃљрЃАрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃџрЃљрЃЊ.

**`mqtt_module.c`-рЃерЃў:**

```c
#include "service_locator.h"
#include "device_identity_interface.h" // РўЁРўЁРўЁ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃљрЃЏ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ

// ...

void connect_to_mqtt_broker() {
    // 1. ­Ъџд рЃЋрЃўрЃдрЃћрЃЉрЃЌ Identity рЃАрЃћрЃарЃЋрЃўрЃАрЃА
    service_handle_t id_service_handle = fmw_service_get("main_identity_service");
    if (!id_service_handle) {
        ESP_LOGE(TAG, "Device Identity Service not found! Cannot generate ClientID.");
        return;
    }

    device_identity_api_t *id_api = (device_identity_api_t *)id_service_handle;

    // 2. ­Ъєћ рЃЋрЃўрЃдрЃћрЃЉрЃЌ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа ID-рЃА
    const char* device_id = id_api->get_device_id();

    // 3. ­Ъцќ рЃЋрЃљрЃцрЃЮрЃарЃЏрЃўрЃарЃћрЃЉрЃЌ ClientID-рЃА
    char client_id_buffer[64];
    snprintf(client_id_buffer, sizeof(client_id_buffer), "synapse-device-%s", device_id);

    // 4. рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ ClientID-рЃА MQTT рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃерЃў
    esp_mqtt_client_config_t mqtt_cfg = {
        .broker.address.uri = "mqtt://...",
        .credentials.client_id = client_id_buffer,
    };

    // ... рЃЋрЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃЌ рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЊрЃљрЃЏрЃДрЃљрЃарЃћрЃЉрЃљрЃА ...
}
```

## 7. РЮЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў

- **рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ `CMakeLists.txt`-рЃерЃў:** `device_identity_service`-рЃўрЃА `CMakeLists.txt` рЃцрЃљрЃўрЃџрЃерЃў, `REQUIRES` рЃАрЃћрЃЦрЃфрЃўрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў `cmd_router_interface`, рЃарЃљрЃЌрЃљ рЃЏрЃљрЃА рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА API-рЃќрЃћ.
- **рЃЊрЃљрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ:** CLI рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ `FMW_EVENT_SYSTEM_START_COMPLETE` рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ `Command Router` рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃљрЃЏ рЃЊрЃарЃЮрЃА рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃарЃљрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃљрЃЊ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ.
