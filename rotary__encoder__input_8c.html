Detailed Description

A universal module to handle rotary encoder inputs using the Task Pool Manager.

Author
Synapse Framework Team
Version
3.0.0
Date
2025-09-07

This module reads rotary encoders and translates their actions into standard SYNAPSE_EVENT_BUTTON_PRESSED events ("UP", "DOWN", "OK"). It no longer creates its own task; instead, it schedules a periodic job with the Shared Task Pool Manager for maximum resource efficiency.
