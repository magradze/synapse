Detailed Description

A module to handle hardware button presses and publish events.

Author
Giorgi Magradze

This module uses GPIO interrupts to detect button presses. To avoid bouncing issues (debounce) and to keep the ISR fast, the ISR only sends the GPIO number to a FreeRTOS queue. A dedicated task then processes the queue, performs debouncing, and publishes a clean FMW_EVENT_BUTTON_PRESSED event.
