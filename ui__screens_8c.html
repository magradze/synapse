Renders a generic button in the footer area (bottom-right).

Renders a generic button in the footer area.

This function dynamically calculates the button's size based on the text length and applies different styling if the button is selected. It is used for both "MENU" and "BACK" buttons to ensure a consistent look.

Parameters
[in]private_dataA pointer to the main UI Manager's private data structure.
[in]textThe text to display inside the button.
[in]is_selectedIf true, the button is rendered with inverted colors.
