# [6.0.0] - 2025-08-21

## Рюе рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў

- **feat(drivers):** ­Ъџї рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ рЃљрЃ«рЃљрЃџрЃў, рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў **`spi_bus`** рЃЏрЃЮрЃЊрЃБрЃџрЃў.
  - **рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ:** рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА SPI рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЉрЃБрЃАрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ, рЃюрЃљрЃЎрЃљрЃЊ-рЃЊрЃљрЃфрЃБрЃџ рЃЊрЃљ рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃБрЃџ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃА.
  - **рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ:** `Resource Manager`-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃћрЃЦрЃАрЃЎрЃџрЃБрЃќрЃўрЃБрЃарЃљрЃЊ рЃЎрЃћрЃбрЃљрЃЋрЃА (lock) SPI рЃ░рЃЮрЃАрЃбрЃА рЃЊрЃљ GPIO рЃърЃўрЃюрЃћрЃЉрЃА, рЃарЃљрЃф рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃљрЃЋрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџ рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃћрЃЉрЃА.
  - **Service API:** рЃљрЃгрЃЋрЃЊрЃўрЃА `spi_bus_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА, рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃю рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃљрЃЋрЃбрЃЮрЃЉрЃБрЃАрЃќрЃћ рЃЊрЃљ рЃерЃћрЃљрЃАрЃарЃБрЃџрЃЮрЃю рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃў.

- **feat(drivers):** ­ЪњЙ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ рЃљрЃ«рЃљрЃџрЃў рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў **`sn74hc595n_writer`**.
  - **рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃърЃўрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ:** рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА `SN74HC595N` shift register-рЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА `spi_bus` рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ.
  - **рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃарЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ:** рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЕрЃўрЃърЃўрЃА рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃќрЃћ рЃњрЃљрЃЊрЃљрЃЉрЃЏрЃљрЃА (daisy-chaining) рЃърЃўрЃюрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃќрЃарЃЊрЃћрЃџрЃљрЃЊ.
  - **Service API:** рЃљрЃгрЃЋрЃЊрЃўрЃА `sn74hc595n_writer_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЉрЃАрЃбрЃарЃљрЃњрЃўрЃарЃћрЃЉрЃА рЃърЃўрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА (`set_pin_state`, `set_chip_pins`).

### ­Ъћё рЃерЃћрЃфрЃЋрЃџрЃўрЃџрЃў

- **refactor(rgb_led):** ­ЪњА **`rgb_led_indicator`** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃЎрЃћрЃЌрЃЊрЃљ.
  - **рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџрЃў рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ:** GPIO рЃърЃўрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљ `ledc` рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ, рЃўрЃА рЃљрЃ«рЃџрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃљрЃ«рЃљрЃџ `sn74hc595n_writer` рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃарЃљрЃф рЃЏрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃўрЃАрЃњрЃљрЃю рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃћрЃџрЃА рЃ«рЃЊрЃўрЃА.
  - **рЃфрЃўрЃцрЃарЃБрЃџ рЃарЃћрЃЪрЃўрЃЏрЃќрЃћ рЃљрЃЊрЃљрЃърЃбрЃљрЃфрЃўрЃљ:** рЃърЃБрЃџрЃАрЃљрЃфрЃўрЃўрЃА (`PULSE`) рЃћрЃцрЃћрЃЦрЃбрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ рЃЊрЃљ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃЊрЃљ рЃфрЃўрЃцрЃарЃБрЃџрЃў (on/off) рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃърЃўрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃљрЃЊрЃљрЃф рЃўрЃА рЃљрЃ«рЃџрЃљ рЃюрЃћрЃџ рЃфрЃўрЃЏрЃфрЃўрЃЏрЃА рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА.

### ­ЪЊё рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

- **docs(drivers):** РюЇ№ИЈ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃАрЃарЃБрЃџрЃў, рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў `README.md` рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃљрЃ«рЃљрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА: `spi_bus` рЃЊрЃљ `sn74hc595n_writer`.
- **docs(interfaces):** ­ЪћЌ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃљрЃ«рЃљрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў (`spi_bus_interface.h`, `sn74hc595n_writer_interface.h`) рЃЊрЃљ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃљ `service_types.h`.
- **docs(rgb_led):** РюЇ№ИЈ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃљ `rgb_led_indicator`-рЃўрЃА `README.md` рЃцрЃљрЃўрЃџрЃў, рЃарЃљрЃЌрЃљ рЃљрЃАрЃљрЃ«рЃљрЃЋрЃЊрЃћрЃА рЃЏрЃўрЃА рЃљрЃ«рЃљрЃџ, shift register-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃА.
