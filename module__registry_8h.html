Detailed Description

მოდულების რეგისტრაციისა და მართვის API

Version
1.0
Date
2025-06-24
Author
Giorgi Magradze

ეს კომპონენტი პასუხისმგებელია სისტემის ყველა მოდულის აღრიცხვასა და მათზე წვდომის უზრუნველყოფაზე. ინახავს მოდულების ცენტრალიზებულ რეესტრს და უზრუნველყოფს ფუნქციებს მათი მოძიების, ინიციალიზაციისა და runtime მართვისთვის.

typedef module_t *(* module_create_fn_t) (const cJSON *config)

მოდულის შემქმნელი (factory) ფუნქციის ტიპი

ყველა მოდულმა უნდა უზრუნველყოს ამ ტიპის ფუნქცია, რომელიც პასუხისმგებელია მოდულის ობიექტის შექმნაზე და მისთვის საჭირო მეხსიერების გამოყოფაზე.

Parameters
[in]configმაჩვენებელი cJSON ობიექტზე, რომელიც შეიცავს ამ მოდულის კონფიგურაციას system_config.json-დან.
Returns
module_t* წარმატების შემთხვევაში აბრუნებს ახლად შექმნილი მოდულის მაჩვენებელს, შეცდომის შემთხვევაში კი NULL-ს.
