Detailed Description

Implementation of the I2C Bus module using the new ESP-IDF v5.x driver.

Author
Giorgi Magradze
Version
2.2.0
Date
2025-09-07

This module provides a thread-safe, resource-managed service for accessing a hardware I2C bus. It utilizes the modern i2c_master.h driver. Resource locking and service registration happen in init, while active operations like scanning happen in start.
