Detailed Description

A module to handle hardware button presses from GPIO or I/O expanders.

Author
Synapse Team
Version
3.0.0
Date
2025-08-27

This module provides a robust and flexible way to handle user button inputs. It supports multiple instances, can operate on direct GPIOs (using interrupts) or via an MCP23017 I/O expander (using polling), and is fully compatible with the framework's dependency injection system. It is designed to be multi-instance safe by avoiding global variables.
