рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃўрЃА рЃцрЃћрЃарЃЌрЃљ рЃАрЃЦрЃћрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃЋрЃћрЃюрЃфрЃўрЃљ

рЃАрЃбрЃљрЃбрЃБрЃАрЃў: рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ (Best Practice) рЃЋрЃћрЃарЃАрЃўрЃљ: 1.0.0

1. ­Ъј» рЃЏрЃўрЃќрЃљрЃюрЃў

рЃћрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА Synapse Framework-рЃерЃў рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃўрЃА рЃбрЃћрЃњрЃћрЃЉрЃўрЃА (TAG) рЃњрЃљрЃцрЃћрЃарЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ рЃАрЃЦрЃћрЃЏрЃљрЃА. рЃљрЃЏ рЃЎрЃЮрЃюрЃЋрЃћрЃюрЃфрЃўрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃљ, рЃарЃљрЃф рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃюрЃљрЃЊ рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљрЃА (debugging).

рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЊрЃљрЃ»рЃњрЃБрЃцрЃћрЃЉрЃљ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА, рЃћрЃарЃЌрЃў рЃерЃћрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃњрЃљрЃљрЃарЃЕрЃўрЃЮрЃА, рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃарЃЮрЃЏрЃћрЃџрЃў рЃюрЃљрЃгрЃўрЃџрЃўрЃЊрЃљрЃю (рЃЏрЃљрЃњ., core, driver, communication) рЃЏрЃЮрЃЊрЃўрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ.

2. РџЎ№ИЈ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃарЃћрЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃўрЃљ logging.h рЃ░рЃћрЃЊрЃћрЃарЃерЃў, DEFINE_COMPONENT_TAG рЃЏрЃљрЃЎрЃарЃЮрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃћрЃЮрЃарЃћ рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃљрЃЊ рЃўрЃдрЃћрЃЉрЃА ANSI рЃцрЃћрЃарЃўрЃА рЃЎрЃЮрЃЊрЃА.

рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃгрЃћрЃАрЃў:

  1. **menuconfig-рЃерЃў рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ:** рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ Component config -> ESP System Settings -> Channel for console output-рЃерЃў рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ [*] Use ANSI escape codes to colorize console output рЃЮрЃцрЃфрЃўрЃљ.

  2. рЃЏрЃљрЃЎрЃарЃЮрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ: рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА .c рЃцрЃљрЃўрЃџрЃерЃў, рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЌ рЃЏрЃљрЃЎрЃарЃЮ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃцрЃћрЃарЃўрЃА рЃЎрЃЮрЃюрЃАрЃбрЃљрЃюрЃбрЃўрЃЌ:

    ```c #include "synapse.h" // рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА logging.h-рЃА

    // рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў communications рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА DEFINE_COMPONENT_TAG("WIFI_MANAGER", SYNAPSE_LOG_COLOR_CYAN); ```

рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃўрЃљ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў рЃцрЃћрЃарЃћрЃЉрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ. рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ, рЃЊрЃљрЃўрЃфрЃЋрЃљрЃЌ рЃћрЃА рЃАрЃЦрЃћрЃЏрЃљ рЃљрЃ«рЃљрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА.
