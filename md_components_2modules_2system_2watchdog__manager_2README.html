1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

Watchdog Manager рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃЌрЃЋрЃўрЃЌрЃљрЃдрЃЊрЃњрЃћрЃюрЃўрЃА рЃЊрЃљ рЃАрЃљрЃўрЃЏрЃћрЃЊрЃЮрЃЮрЃЉрЃўрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў. рЃЏрЃўрЃАрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃљ, рЃЊрЃљрЃўрЃфрЃЋрЃљрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃА рЃњрЃљрЃГрЃћрЃЊрЃЋрЃўрЃАрЃњрЃљрЃю (software freeze/hang).

рЃбрЃарЃљрЃЊрЃўрЃфрЃўрЃБрЃџрЃў watchdog-рЃћрЃЉрЃўрЃАрЃњрЃљрЃю рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃўрЃЌ, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃљрЃЊ "рЃўрЃЎрЃЋрЃћрЃЉрЃћрЃЉрЃўрЃљрЃю", рЃћрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃ«рЃЊрЃћрЃюрЃА **"рЃГрЃЎрЃЋрЃўрЃљрЃюрЃў" рЃљрЃюрЃБ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў (application-aware)** watchdog-рЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА ESP-IDF-рЃўрЃА Task Watchdog Timer-рЃА (TWDT), рЃЏрЃљрЃњрЃарЃљрЃЏ "рЃљрЃЎрЃЋрЃћрЃЉрЃћрЃЉрЃА" рЃЏрЃљрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃБ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў, рЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў рЃњрЃљрЃЏрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ рЃЊрЃљ рЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃўрЃарЃћрЃЉрЃА.

2. ­ЪЈЏ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

  • рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃБрЃџрЃў "рЃњрЃБрЃџрЃўрЃАрЃфрЃћрЃЏрЃљ" (Distributed Heartbeat): рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃа рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃљ рЃћрЃарЃЌ рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃБрЃа рЃбрЃљрЃАрЃЎрЃА. рЃљрЃЏрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ, рЃўрЃА рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ "рЃњрЃБрЃџрЃўрЃАрЃфрЃћрЃЏрЃўрЃА" рЃАрЃўрЃњрЃюрЃљрЃџрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃАрЃњрЃљрЃю. Watchdog-рЃў "рЃўрЃЎрЃЋрЃћрЃЉрЃћрЃЉрЃљ" рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃфрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃАрЃњрЃљрЃю рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃўрЃњрЃюрЃљрЃџрЃў.
  • Service-driven: рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃгрЃЋрЃЊрЃўрЃА watchdog_api_t рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА, рЃЊрЃљрЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃЮрЃю рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЌрЃљрЃЋрЃў, рЃарЃЮрЃњрЃЮрЃарЃф "рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў" рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў рЃЊрЃљ рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃљрЃЊ рЃњрЃљрЃљрЃњрЃќрЃљрЃЋрЃюрЃЮрЃю "рЃњрЃБрЃџрЃўрЃАрЃфрЃћрЃЏрЃљ".
  • рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃў рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ: рЃфрЃљрЃџрЃЎрЃћ FreeRTOS рЃбрЃљрЃАрЃЎрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ, рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА System Timer Service-рЃА, рЃарЃљрЃЌрЃљ рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃЮрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ. рЃћрЃА рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљрЃА рЃЊрЃљ рЃќрЃЮрЃњрЃљрЃЋрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃА.
  • рЃцрЃћрЃюрЃћрЃЉрЃљрЃЊ рЃЊрЃљрЃДрЃЮрЃцрЃўрЃџрЃў рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃџрЃЮрЃЉрЃљ:
    1. ESP-IDF TWDT: рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃбрЃљрЃўрЃЏрЃћрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃЋрЃћрЃЋрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџ рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљрЃА.
    2. **watchdog_manager:** рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃгрЃДрЃЋрЃћрЃбрЃА, "рЃЋрЃљрЃГрЃљрЃЏрЃЮрЃЌ" рЃЌрЃБ рЃљрЃарЃљ TWDT-рЃА.
    3. рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў: рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃюрЃў рЃљрЃарЃўрЃљрЃю, рЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃњрЃљрЃљрЃњрЃќрЃљрЃЋрЃюрЃЮрЃю рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў "рЃњрЃБрЃџрЃўрЃАрЃфрЃћрЃЏрЃљ".

3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф Kconfig-рЃўрЃА, рЃўрЃАрЃћ system_config.json рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

**system_config.json рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

"enabled": true,

}

рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:

рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃбрЃўрЃърЃў рЃљрЃдрЃгрЃћрЃарЃљ Default (Kconfig)
instance_name рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў. main_watchdog
check_interval_ms рЃарЃўрЃфрЃ«рЃЋрЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃў (рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃћрЃЉрЃерЃў), рЃарЃљ рЃАрЃўрЃ«рЃерЃўрЃарЃўрЃЌрЃљрЃф рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџ "рЃњрЃБрЃџрЃўрЃАрЃфрЃћрЃЏрЃћрЃЉрЃА". 2000
heartbeat_timeout_ms рЃарЃўрЃфрЃ«рЃЋрЃў рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЊрЃарЃЮ (рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃћрЃЉрЃерЃў), рЃарЃЮрЃЏрЃџрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃўрЃф рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ "рЃњрЃБрЃџрЃўрЃАрЃфрЃћрЃЏрЃљрЃА" рЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃАрЃњрЃљрЃю. 5000

4. ­Ъћї Service API (<tt>watchdog_api_t</tt>)
