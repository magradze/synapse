Detailed Description

Monitors system health using the Shared Task Pool Manager.

Author
Synapse Framework Team
Version
2.0.0
Date
2025-09-07

This module periodically checks system resources (heap, task stacks) by scheduling a recurring job with the Task Pool Manager, thus avoiding the need for its own dedicated task.
