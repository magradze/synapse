module_t * logger_module_create ( const cJSON *  config)

Creates a new Logger Module instance.

This function creates and initializes a new logger module instance that subscribes to all events on the Event Bus and outputs debugging information to the serial console. The module supports runtime control through framework command events.

Parameters
[in]configJSON configuration object containing module settings. Expected fields:
  • "instance_name" (string): Name of this logger instance (default: "main_logger")
Returns
module_t* Pointer to the created module instance, or NULL on failure.
Note
The module will automatically subscribe to all system events through the System Manager's universal event subscription mechanism.
This module is typically disabled in production builds to avoid excessive logging output.
See also
base_module.h for module interface details
event_bus.h for event subscription mechanism
