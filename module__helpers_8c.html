Detailed Description

მოდულების დამხმარე (Helper) ფუნქციების იმპლემენტაცია.

Version
1.1
Date
2025-06-27
Author
Giorgi Magradze

ეს ფაილი შეიცავს სტანდარტულ, მრავალჯერად ფუნქციებს, რომლებიც ამარტივებენ მოდულების სიცოცხლის ციკლის საბაზისო ოპერაციების (enable, disable, get_status) იმპლემენტაციას. ყველა ოპერაცია არის ნაკად-უსაფრთხო (thread-safe).
