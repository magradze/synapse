Detailed Description

A centralized service for scheduling event publishing.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-07-06

This module provides a centralized service that allows other modules to schedule one-shot or periodic events to be posted on the Event Bus. It abstracts the underlying esp_timer mechanism, providing a simpler, decoupled, and resource-efficient way to handle timed tasks.
