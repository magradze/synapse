Detailed Description

SD Card (FAT filesystem) backend for the Storage Manager.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-07-05

This file provides the concrete implementation of the storage_api_t interface using the ESP-IDF FAT filesystem driver for SD cards over an SPI bus. It supports all file and key-value operations.
