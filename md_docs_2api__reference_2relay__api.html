1. Рїе№ИЈ CLI (Command Line Interface)

    • <action>: рЃерЃћрЃАрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃћрЃџрЃў рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ. рЃерЃћрЃАрЃљрЃФрЃџрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃљ:
