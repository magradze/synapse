­ЪЊ▓ рЃЏрЃЮрЃЊрЃБрЃџрЃў <tt>ble_provisioning</tt>

1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў: ble_provisioning
рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ: provisioning
рЃЋрЃћрЃарЃАрЃўрЃљ: 2.4.0 (рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃўрЃљ)
рЃљрЃЋрЃбрЃЮрЃарЃў: Synapse Framework Team

ble_provisioning рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА (provisioning) рЃърЃарЃЮрЃфрЃћрЃАрЃА Bluetooth Low Energy (BLE)-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА ESP-IDF-рЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ wifi_provisioning рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃА, рЃарЃљрЃЌрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃЏ рЃерЃћрЃФрЃџрЃЮрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃЊ рЃЏрЃўрЃўрЃдрЃЮрЃА WiFi рЃЦрЃАрЃћрЃџрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў (SSID рЃЊрЃљ рЃърЃљрЃарЃЮрЃџрЃў) рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃЊрЃљрЃю (рЃЏрЃљрЃњ., Espressif-рЃўрЃА ESP BLE Prov).

рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљрЃљ BLE рЃАрЃљрЃарЃћрЃЎрЃџрЃљрЃЏрЃЮ рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа ID-рЃА device_identity_service-рЃўрЃЊрЃљрЃю, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа рЃАрЃљрЃ«рЃћрЃџрЃА рЃЦрЃАрЃћрЃџрЃерЃў.

2. ­ЪЈЌ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

  • рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў (Service-driven): рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА ble_prov_api_t рЃАрЃћрЃарЃЋрЃўрЃАрЃА Service Locator-рЃерЃў. рЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА, рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃю, рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ provisioning-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў, рЃљрЃю рЃЊрЃљрЃўрЃгрЃДрЃЮрЃю рЃўрЃА рЃўрЃФрЃБрЃџрЃћрЃЉрЃўрЃЌ.
  • рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃќрЃћ-рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў (Event-driven): рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃЦрЃбрЃўрЃБрЃарЃљрЃЊ рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА Event Bus-рЃЌрЃљрЃю:
    • рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ PROV_STARTED, PROV_ENDED рЃЊрЃљ PROV_CREDENTIALS_RECEIVED, рЃарЃљрЃЌрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃАрЃ«рЃЋрЃљ рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃЏрЃљ рЃўрЃфрЃЮрЃЊрЃюрЃћрЃю рЃЏрЃўрЃАрЃў рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.
    • рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ WIFI_EVENT_IP_ASSIGNED (рЃарЃљрЃЌрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃљрЃАрЃарЃБрЃџрЃЮрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў) рЃЊрЃљ PROV_START_REQUESTED (рЃарЃљрЃЌрЃљ рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃњрЃљрЃарЃћ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃЌ).
  • **рЃЏрЃљрЃдрЃљрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃћ (init_level: 90):** рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў (Consumer), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ device_identity_service-рЃќрЃћ (init_level: 20), ble_provisioning рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃњрЃЋрЃўрЃљрЃю рЃћрЃбрЃљрЃърЃќрЃћ. рЃћрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃАрЃћрЃарЃЋрЃўрЃАрЃў рЃњрЃљрЃарЃљрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃљрЃЊ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃЏрЃўрЃАрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљрЃЏрЃЊрЃћ.
  • рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ (Service Dependency): рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ device_identity_service-рЃќрЃћ, рЃарЃљрЃЌрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў ID. рЃћрЃА рЃљрЃарЃўрЃА рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА "рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃџрЃў-рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА" рЃърЃљрЃбрЃћрЃарЃюрЃўрЃА рЃЎрЃџрЃљрЃАрЃўрЃЎрЃБрЃарЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў.

3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ Kconfig-рЃўрЃА (idf.py menuconfig) рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

**Kconfig рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

Component config -> Synapse Framework -> Modules -> BLE Provisioning рЃЏрЃћрЃюрЃўрЃБрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ:

  • **BLE Device Name:** рЃАрЃљрЃарЃћрЃЎрЃџрЃљрЃЏрЃЮ рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃў (Prefix), рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЊрЃљрЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў ID. рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃАрЃљрЃ«рЃћрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ: Prefix-DeviceID (рЃЏрЃљрЃњ., Synapse-5C013B4A9954).
  • **Proof-of-Possession (PoP) String:** рЃАрЃљрЃўрЃЊрЃБрЃЏрЃџрЃЮ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА Security 1 рЃарЃћрЃЪрЃўрЃЏрЃерЃў.
  • **Security Version:** рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљ (0 - рЃЊрЃљрЃБрЃерЃўрЃцрЃарЃљрЃЋрЃў, 1 - рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў PoP-рЃўрЃЌ).

**config.json рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

config.json-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ Kconfig-рЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃцрЃљрЃарЃЋрЃљ. рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ: device_name рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃљрЃдрЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃАрЃљрЃ«рЃћрЃџрЃў рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃљрЃЊ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃЊрЃћрЃЉрЃљ.

4. ­ЪЏа№ИЈ Service API (<tt>ble_prov_api_t</tt>)

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљрЃф рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ fmw_service_get-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃўрЃЌ, config.json-рЃерЃў рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў instance_name-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ (рЃЏрЃљрЃњ., "main_ble_provisioning").
