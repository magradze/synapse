Assertions рЃЊрЃљ Guard Clauses рЃЎрЃЮрЃюрЃЋрЃћрЃюрЃфрЃўрЃљ

рЃАрЃбрЃљрЃбрЃБрЃАрЃў: рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ (Best Practice) рЃЋрЃћрЃарЃАрЃўрЃљ: 1.0.0

1. ­Ъј» рЃЏрЃўрЃќрЃљрЃюрЃў

рЃћрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА Synapse Framework-рЃерЃў рЃЎрЃЮрЃЊрЃўрЃА рЃЏрЃЊрЃњрЃарЃљрЃЊрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў рЃЮрЃарЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃљрЃбрЃћрЃарЃюрЃўрЃА Рђћ Assertions (SYNAPSE_ASSERT) рЃЊрЃљ Guard Clauses (SYNAPSE_GUARD) Рђћ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ рЃЎрЃЮрЃюрЃЋрЃћрЃюрЃфрЃўрЃљрЃА.

рЃљрЃЏ рЃърЃљрЃбрЃћрЃарЃюрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЎрЃЮрЃЊрЃА рЃ«рЃЊрЃўрЃА рЃБрЃцрЃарЃЮ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃА, рЃЌрЃЋрЃўрЃЌ-рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃљрЃЊрЃА рЃЊрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃюрЃљрЃЊ рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃљрЃА рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃћрЃбрЃљрЃърЃќрЃћ.

2. ­Ъцћ рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ: Assertion vs. Error Handling

рЃАрЃљрЃюрЃљрЃЏ рЃЏрЃљрЃЎрЃарЃЮрЃћрЃЉрЃА рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃљрЃЋрЃЌ, рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ, рЃњрЃљрЃЋрЃўрЃљрЃќрЃарЃЮрЃЌ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ рЃљрЃЏ рЃЮрЃа рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃљрЃА рЃерЃЮрЃарЃўрЃА:

  • Error Handling (рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ):
    • рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА? рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЏрЃЮрЃ«рЃЊрЃћрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃюрЃЮрЃарЃЏрЃљрЃџрЃБрЃарЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА (runtime). рЃћрЃА рЃљрЃарЃўрЃА рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў, рЃЌрЃБрЃЏрЃфрЃљ рЃљрЃарЃљрЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃў.
    • рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў: malloc-рЃЏрЃљ рЃЋрЃћрЃа рЃњрЃљрЃЏрЃЮрЃДрЃЮ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ, I2C рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ рЃљрЃа рЃърЃљрЃАрЃБрЃ«рЃЮрЃЉрЃА, рЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ.
    • рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў: if (...) { ... return ESP_ERR_...; } рЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃў.
    • рЃерЃћрЃЊрЃћрЃњрЃў: рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃљрЃЎрЃБрЃарЃљрЃбрЃБрЃџрЃљрЃЊ рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА рЃЊрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф рЃгрЃћрЃАрЃў, рЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.
  • Assertion (рЃЏрЃбрЃЎрЃўрЃфрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ):
    • рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА? рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃўрЃЌ рЃљрЃарЃљрЃАрЃЊрЃарЃЮрЃА, рЃљрЃарЃљрЃюрЃљрЃўрЃа рЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљрЃерЃў рЃљрЃа рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃфрЃЊрЃљрЃарЃў.
    • рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў: рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃА рЃњрЃљрЃЊрЃљрЃћрЃфрЃљ NULL рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃћрЃџрЃў, рЃАрЃљрЃЊрЃљрЃф рЃћрЃА рЃЊрЃљрЃБрЃерЃЋрЃћрЃЉрЃћрЃџрЃўрЃљ; рЃЏрЃљрЃАрЃўрЃЋрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў рЃњрЃљрЃЋрЃўрЃЊрЃљ рЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃА рЃњрЃљрЃарЃћрЃЌ; switch рЃЉрЃџрЃЮрЃЎрЃерЃў рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃљ default рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃљрЃЊ рЃЏрЃўрЃБрЃдрЃгрЃћрЃЋрЃљрЃЊрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА.
    • рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў: SYNAPSE_ASSERT(...).
    • **рЃерЃћрЃЊрЃћрЃњрЃў (debug рЃарЃћрЃЪрЃўрЃЏрЃерЃў):** рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЊрЃљрЃБрЃДрЃЮрЃЋрЃюрЃћрЃЉрЃџрЃўрЃЋ рЃЕрЃћрЃарЃЊрЃћрЃЉрЃљ (panic), рЃарЃљрЃЊрЃњрЃљрЃю рЃЏрЃўрЃАрЃў рЃерЃўрЃЊрЃљ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ рЃБрЃЎрЃЋрЃћ рЃЎрЃЮрЃарЃБрЃЏрЃърЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃљ рЃАрЃљрЃ«рЃўрЃцрЃљрЃЌрЃЮрЃљ.

‍рЃЮрЃЦрЃарЃЮрЃА рЃгрЃћрЃАрЃў: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ if рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ SYNAPSE_ASSERT Рђћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃЉрЃљрЃњрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃџрЃћрЃюрЃљрЃЊ.

3. <tt>SYNAPSE_ASSERT</tt> рЃЏрЃљрЃЎрЃарЃЮ

SYNAPSE_ASSERT рЃљрЃарЃўрЃА рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃцрЃљрЃќрЃљрЃерЃў рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЏрЃћрЃюрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃфрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

3.1. рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ?

  • рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃў, рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА (NULL рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА) рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ.
  • рЃЎрЃЮрЃЊрЃўрЃА рЃўрЃЏ рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃерЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃљрЃЊ рЃЏрЃўрЃБрЃдрЃгрЃћрЃЋрЃљрЃЊрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА.
  • рЃерЃўрЃЊрЃљ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА (internal state) рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљрЃЏрЃЊрЃћ.

3.2. рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў
