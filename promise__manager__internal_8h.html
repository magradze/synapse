Creates a new promise with pre-registered callbacks.

Called by a service provider at the start of an async operation.

Parameters
[in]then_cbThe function to call when the promise is resolved.
[in]catch_cbThe function to call when the promise is rejected.
[in]user_contextA context pointer to be passed to the callbacks.
Returns
A new promise_handle_t, or NULL on failure.
