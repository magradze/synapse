Detailed Description

მოდულებისთვის განკუთვნილი ზოგადი დამხმარე (Helper) ფუნქციები.

Version
1.0
Date
2025-06-26
Author
Giorgi Magradze

ეს ფაილი შეიცავს მოდულების სიცოცხლის ციკლის სამართავად განკუთვნილ ზოგად იმპლემენტაციებს. ის მიზნად ისახავს, შეამციროს კოდის დუბლირება იმ მოდულებში, რომლებსაც აქვთ სტანდარტული enable, disable ან get_status ლოგიკა.
