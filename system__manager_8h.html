Detailed Description

სისტემის მენეჯერის Public API.

ეს ფაილი განსაზღვრავს ფრეიმვორქის ინიციალიზაციის, გაშვების და runtime მართვის მთავარ ფუნქციებს. System Manager მოქმედებს როგორც "დირიჟორი", რომელიც ორკესტრირებას უწევს ბირთვის კომპონენტებისა და მოდულების მუშაობას.

ESP_FAILთუ რომელიმე კრიტიკული კომპონენტის ინიციალიზაცია ვერ მოხერხდა.
