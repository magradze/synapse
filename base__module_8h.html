Detailed Description

საბაზისო მოდულის ინტერფეისის და სტატუსის ტიპების აღწერა.

Version
2.0
Date
2025-06-26
Author
Giorgi Magradze

ეს ფაილი განსაზღვრავს მოდულის სტატუსის ენუმერაციას, საბაზისო ოპერაციების function pointer-ებს, საბაზისო მოდულის ინტერფეისის სტრუქტურას და მოდულის მთავარ სტრუქტურას. უზრუნველყოფს მოდულების ერთიან კონტრაქტს სისტემაში რეგისტრაციის, ინიციალიზაციის, გაშვების, გაჩერების, ივენთების დამუშავების, enable/disable/reconfigure და სტატუსის მიღებისათვის.
