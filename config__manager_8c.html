Detailed Description

Implementation of the Configuration Manager.

Author
Giorgi Magradze & Synapse AI
Version
7.0.0 - Final Correct Implementation
Date
2025-07-01

This is the final, correct implementation that supports the decentralized config.json model. It assembles the full configuration from multiple embedded files on first boot and then uses a simple, single-key NVS strategy for subsequent boots.

esp_err_t synapse_config_manager_init ( void  )

ახდენს კონფიგურაციის მენეჯერის ინიციალიზაციას.

ეს ფუნქცია კითხულობს კონფიგურაციას NVS-დან. თუ NVS-ში კონფიგურაცია არ არსებობს, ის ტვირთავს system_config.json-ის default ვერსიას ფაილური სისტემიდან. ეს ფუნქცია უნდა გამოიძახოს system_manager-მა ინიციალიზაციისას.

Returns
esp_err_t ოპერაციის წარმატების კოდი.
Return values
ESP_OKთუ კონფიგურაცია წარმატებით ჩაიტვირთა.
ESP_FAILთუ ჩატვირთვა ვერ მოხერხდა.
