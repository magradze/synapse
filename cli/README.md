# Synapse CLI (v1.0.0)

`synapse-cli` рЃљрЃарЃўрЃА рЃћрЃарЃЌрЃўрЃљрЃюрЃў, рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃ«рЃљрЃќрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ Synapse ESP Framework-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА, рЃерЃћрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃџрЃљрЃЊ. рЃўрЃА рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљрЃА рЃБрЃЎрЃћрЃЌрЃћрЃЉрЃА `idf.py`-рЃА, рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃбрЃЮрЃарЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃАрЃЌрЃљрЃю рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃўрЃА рЃарЃБрЃбрЃўрЃюрЃБрЃџ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃА.

## ­Ъј» рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў

- **рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ:** рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃБрЃцрЃЌрЃљ рЃЉрЃўрЃџрЃЊрЃў, рЃцрЃџрЃћрЃерЃќрЃћ рЃЕрЃљрЃгрЃћрЃарЃљ рЃЊрЃљ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў рЃћрЃарЃЌрЃўрЃљрЃюрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃЌ.
- **рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ:** рЃљрЃ«рЃљрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА "рЃЕрЃЮрЃюрЃЕрЃ«рЃўрЃА" рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЦрЃЏрЃюрЃљ рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЊрЃљрЃфрЃЋрЃўрЃЌ.
- **рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ:** рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃерЃћрЃџрЃў (Shell) рЃЊрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃерЃў рЃАрЃћрЃарЃўрЃБрЃџрЃў рЃърЃЮрЃарЃбрЃўрЃЌ.
- **рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ (рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃЮрЃЊ):** `system_config.json`-рЃўрЃА рЃАрЃўрЃАрЃгрЃЮрЃарЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ.

## ­ЪњЙ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

`synapse-cli` рЃљрЃарЃўрЃА Python-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў. рЃЏрЃўрЃА рЃЊрЃљрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ, рЃЏрЃўрЃ░рЃДрЃћрЃЋрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃА:

1. **рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃЦрЃЋрЃЌ Python рЃЊрЃљ pip:**

    ```bash
    python3 --version
    pip3 --version
    ```

2. **рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ CLI рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃерЃў:**
    рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `cli/` рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ. рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЉрЃА `synapse-cli`-рЃА рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃА (`click`, `rich`, `pyfiglet` рЃЊрЃљ рЃљ.рЃе.).

    `-e` (`editable`) рЃцрЃџрЃљрЃњрЃў рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЎрЃЮрЃЊрЃерЃў рЃерЃћрЃбрЃљрЃюрЃўрЃџрЃў рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃљрЃЊ рЃљрЃўрЃАрЃљрЃ«рЃћрЃЉрЃљ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃќрЃћ рЃ«рЃћрЃџрЃљрЃ«рЃљрЃџрЃў рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.

    ```bash
    pip3 install -e .
    ```

3. **рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ:**
    рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ **рЃљрЃ«рЃљрЃџрЃў** рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў рЃЊрЃљ рЃљрЃЎрЃарЃўрЃцрЃћрЃЌ:

    ```bash
    synapse --help
    ```

    рЃЌрЃЦрЃЋрЃћрЃю рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃЮрЃЌ `synapse-cli`-рЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃўрЃБ, рЃџрЃљрЃЏрЃљрЃќрЃў рЃцрЃћрЃарЃћрЃЉрЃўрЃЌрЃљ рЃЊрЃљ рЃАрЃбрЃўрЃџрЃћрЃЉрЃўрЃЌ.

4. **рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ:**
    рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ **рЃљрЃ«рЃљрЃџрЃў** рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў рЃЊрЃљ рЃљрЃЎрЃарЃўрЃцрЃћрЃЌ:

    ```bash
    synapse --help
    ```

    рЃЌрЃЦрЃЋрЃћрЃю рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃЮрЃЌ `synapse-cli`-рЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃўрЃБ.

## ­Ъџђ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

`synapse-cli` рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `рЃ»рЃњрЃБрЃцрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ [рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў] --рЃЮрЃцрЃфрЃўрЃћрЃЉрЃў` рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА.

### 1. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ (`project` рЃ»рЃњрЃБрЃцрЃў)

- **рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЉрЃўрЃџрЃЊрЃў:**

  ```bash
  synapse project build
  ```

  - **`--reconfigure` рЃцрЃџрЃљрЃњрЃўрЃЌ:** рЃ»рЃћрЃа рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА `reconfigure`-рЃА, рЃерЃћрЃЏрЃЊрЃћрЃњ `build`-рЃА.

    ```bash
    synapse project build --reconfigure
    ```

- **Firmware-рЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ:**

  ```bash
  synapse project flash
  ```

  - **рЃърЃЮрЃарЃбрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃўрЃЌ:**

    ```bash
    synapse project flash -p /dev/ttyUSB0
    ```

- **рЃАрЃћрЃарЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃў:**

  ```bash
  synapse project monitor
  ```

- **рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ:**

  ```bash
  synapse project clean
  ```

### 2. рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ (`module` рЃ»рЃњрЃБрЃцрЃў)

- **рЃљрЃ«рЃљрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ:**

  ```bash
  synapse module create "<Module Name>" -t <category> -d "<description>"
  ```

  - **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

    ```bash
    synapse module create "MQTT Client" -t communications -d "Handles MQTT connection and messaging"
    ```

### 3. рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃАрЃЌрЃљрЃю рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ (`device` рЃ»рЃњрЃБрЃцрЃў)

рЃћрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю `Command Router` рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃџ рЃАрЃћрЃарЃўрЃБрЃџ рЃерЃћрЃџрЃА.

- **рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃерЃћрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ:**

  ```bash
  synapse device shell
  ```

  - рЃњрЃљрЃАрЃЋрЃџрЃљ: `Ctrl + ]`

- **рЃћрЃарЃЌрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ:**

  ```bash
  synapse device cmd <command_string>
  ```

  - **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

    ```bash
    synapse device cmd health
    synapse device cmd relay main_light on
    ```

- **`health` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ:**

  ```bash
  synapse device health
  ```

### 4. рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ (`config` рЃ»рЃњрЃБрЃцрЃў)

- **рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ:**
  рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА `system_config.json`-рЃА рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃБрЃа рЃАрЃўрЃАрЃгрЃЮрЃарЃћрЃќрЃћ рЃЊрЃљ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў (рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃўрЃљ рЃЏрЃљрЃЌ `module.json` рЃцрЃљрЃўрЃџрЃћрЃЉрЃерЃў).
  
  ```bash
  synapse config validate
  ```

- **рЃАрЃ«рЃЋрЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃўрЃЌ:**

  ```bash
  synapse config validate -f my_custom_config.json
  ```

## ­ЪћД рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃњрЃЋрЃљрЃарЃћрЃЉрЃљ

- **`command not found: synapse`:**
  - рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ `pip`-рЃўрЃА рЃЏрЃўрЃћрЃа рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ (`~/.local/bin` Linux/macOS-рЃќрЃћ) рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА `PATH` рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃерЃў.
  - рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ рЃЊрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃњрЃљрЃ«рЃАрЃюрЃљ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ.

- **`Error: 'idf.py' not found...`:**
  - рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ ESP-IDF-рЃўрЃА рЃњрЃљрЃарЃћрЃЏрЃЮ рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЦрЃЋрЃЌ (`. ./export.sh`) рЃўрЃЏ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў, рЃАрЃљрЃЊрЃљрЃф `synapse` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА рЃБрЃерЃЋрЃћрЃЉрЃЌ. `synapse-cli` рЃерЃўрЃЊрЃљ рЃЊрЃЮрЃюрЃћрЃќрЃћ рЃўрЃФрЃљрЃ«рЃћрЃЉрЃА `idf.py`-рЃА рЃЊрЃљ рЃЏрЃљрЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ ESP-IDF-рЃўрЃА рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃќрЃћ.
