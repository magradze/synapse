Detailed Description

Public Service API contract for the RGB LED Indicator module.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-07-01

This header defines the abstract interface (API structure) for the RGB LED Indicator service. Any module wishing to control an RGB LED via the Service Locator should include this file. This ensures that the consumer module is decoupled from the specific implementation of the RGB LED driver.
