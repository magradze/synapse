# ­ЪЊ║ рЃЏрЃЮрЃЊрЃБрЃџрЃў: `ssd1306_oled_display`

**­ЪЌѓ№ИЈ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ:** `displays`  
**Рюњ№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃў:** Giorgi Magradze & Synapse AI Expert  
**­Ъћќ рЃЋрЃћрЃарЃАрЃўрЃљ:** 2.1.0

## 1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

`ssd1306_oled_display` рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА **рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА SSD1306-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃЮрЃЦрЃарЃЮрЃЏрЃБрЃџрЃў OLED рЃћрЃЎрЃарЃљрЃюрЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃцрЃљрЃАрЃЮрЃЋрЃљрЃю рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА. рЃўрЃА рЃљрЃа рЃљрЃарЃўрЃА рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў; рЃћрЃА рЃљрЃарЃўрЃА **рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃЮ рЃЊрЃљрЃцрЃљ**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЋрЃбрЃЮрЃюрЃЮрЃЏрЃўрЃБрЃарЃљрЃЊ рЃљрЃАрЃљрЃ«рЃљрЃЋрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџ рЃАрЃбрЃљрЃбрЃБрЃАрЃћрЃЉрЃА рЃЊрЃљ, рЃљрЃЏрЃљрЃЋрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ, рЃљрЃгрЃЋрЃЊрЃўрЃА рЃЏрЃФрЃџрЃљрЃЋрЃа API-рЃА рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА, рЃарЃљрЃЌрЃљ рЃЏрЃљрЃЌ рЃерЃћрЃФрЃџрЃЮрЃю рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљ.

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃўрЃДрЃЮрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў, рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃарЃљ рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџ FreeRTOS рЃбрЃљрЃАрЃЎрЃА рЃЊрЃљ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃарЃўрЃњрЃА (Queue) рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃљрЃарЃљрЃЉрЃџрЃЮрЃЎрЃўрЃарЃћрЃЉрЃљрЃЊрЃў, рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## 2. ­ЪЈЏ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

- **­ЪцЮ рЃ░рЃўрЃЉрЃарЃўрЃЊрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃљрЃарЃўрЃА **Service Provider** рЃЊрЃљ **Event Consumer**.
  - **рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃљ:** рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА `ssd1306_api_t` рЃАрЃћрЃарЃЋрЃўрЃАрЃА `Service Locator`-рЃерЃў, рЃарЃљрЃф рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃгрЃћрЃарЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа, рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА.
  - **рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ:** рЃњрЃљрЃЏрЃЮрЃўрЃгрЃћрЃарЃА рЃЊрЃљ рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃа рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА (`FMW_EVENT_WIFI_STATUS_READY`, `FMW_EVENT_AGGREGATED_SENSOR_REPORT`), рЃарЃљрЃЌрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃЮрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ.
- **­Ъћё рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ-рЃърЃљрЃАрЃБрЃ«рЃўрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў:** WiFi рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃАрЃљрЃўрЃЏрЃћрЃЊрЃЮ рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃа рЃћрЃџрЃЮрЃЊрЃћрЃЉрЃљ рЃърЃљрЃАрЃўрЃБрЃарЃљрЃЊ рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃА. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `System Timer Service`-рЃА, рЃарЃљрЃЌрЃљ рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃљрЃЊ рЃњрЃљрЃљрЃњрЃќрЃљрЃЋрЃюрЃЮрЃА `"wifi status --silent"` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃАрЃљрЃърЃљрЃАрЃБрЃ«рЃЮрЃЊ рЃЏрЃЮрЃАрЃБрЃџ `FMW_EVENT_WIFI_STATUS_READY` рЃўрЃЋрЃћрЃюрЃЌрЃА.
- **­Ъќ╝№ИЈ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЊрЃљрЃДрЃЮрЃцрЃљ:** рЃћрЃЎрЃарЃљрЃюрЃў рЃЊрЃљрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ рЃЮрЃа рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃќрЃЮрЃюрЃљрЃЊ:
  - **Header (рЃќрЃћрЃЊрЃљ рЃќрЃЮрЃџрЃў):** рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃџрЃЮрЃњрЃЮ, WiFi рЃАрЃбрЃљрЃбрЃБрЃАрЃў).
  - **Content Area (рЃерЃўрЃюрЃљрЃљрЃарЃАрЃўрЃА рЃљрЃарЃћ):** рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў, рЃЏрЃћрЃюрЃўрЃБрЃћрЃЉрЃў).
- **­ЪЊЅ рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљ:** рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃАрЃарЃБрЃџрЃљрЃЊ рЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃљ `i2c_bus` рЃАрЃћрЃарЃЋрЃўрЃАрЃА I2C рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃљрЃф рЃЏрЃљрЃА рЃљрЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃћрЃЉрЃА рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃќрЃћ "рЃцрЃўрЃЦрЃарЃўрЃАрЃњрЃљрЃю".
- **­Ъџђ рЃЏрЃљрЃдрЃљрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃћ (`init_level: 70`):** рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃњрЃЋрЃўрЃљрЃю, рЃарЃљрЃЌрЃљ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЋрЃўрЃДрЃЮрЃЌ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА (`i2c_bus`, `system_timer`), рЃБрЃЎрЃЋрЃћ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ.

## 3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ (`config.json`)

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃЏрЃўрЃАрЃў `config.json` рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

**рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

```json
[
  {
    "type": "ssd1306_oled_display",
    "enabled": true,
    "config": {
      "instance_name": "main_display",
      "i2c_bus_instance": "main_i2c_bus",
      "i2c_addr": "0x3C",
      "width": 128,
      "height": 64,
      "init_text": "Synapse"
    }
  }
]
```

**рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

| рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў | рЃбрЃўрЃърЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ |
| :--- | :--- | :--- | :---: |
| `instance_name` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў. рЃљрЃЏ рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ рЃ«рЃЊрЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ. | РюЁ |
| `i2c_bus_instance` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃўрЃЏ `i2c_bus` рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА `instance_name`, рЃарЃЮрЃЏрЃћрЃџрЃќрЃћрЃф рЃћрЃЎрЃарЃљрЃюрЃўрЃљ рЃЏрЃўрЃћрЃарЃЌрЃћрЃЉрЃБрЃџрЃў. | РюЁ |
| `i2c_addr` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА 7-рЃЉрЃўрЃбрЃўрЃљрЃюрЃў I2C рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў (рЃ░рЃћрЃЦрЃАрЃљрЃЊрЃћрЃфрЃўрЃЏрЃљрЃџрЃБрЃа рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў, рЃЏрЃљрЃњ., `"0x3C"`). | РюЁ |
| `width` | рЃарЃўрЃфрЃ«рЃЋрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃћ рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў. | РЮї (Default: 128) |
| `height` | рЃарЃўрЃфрЃ«рЃЋрЃў | рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў. | РЮї (Default: 64) |
| `init_text` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў | рЃбрЃћрЃЦрЃАрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃљрЃА (Splash Screen). | РЮї (Default: "Synapse") |

## 4. ­Ъћї Service API (`ssd1306_api_t`)

рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃю рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃљрЃЏ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ `Service Locator`-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ. рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃбрЃўрЃърЃў: **`FMW_SERVICE_TYPE_DISPLAY_API`**.

**API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:**

- **`esp_err_t enable(context);`**  
  ­ЪЪб рЃарЃЌрЃљрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃА рЃЊрЃљ рЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃўрЃюрЃћрЃЉрЃА рЃЏрЃўрЃА рЃцрЃЮрЃюрЃБрЃа рЃбрЃљрЃАрЃЎрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.
- **`esp_err_t disable(context);`**  
  РџФ рЃЌрЃўрЃерЃљрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃА рЃЊрЃљ рЃљрЃърЃљрЃБрЃќрЃћрЃЉрЃА рЃЏрЃўрЃА рЃцрЃЮрЃюрЃБрЃа рЃбрЃљрЃАрЃЎрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃќрЃЮрЃњрЃљрЃЊ.
- **`esp_err_t clear(context);`**  
  Рюе рЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃА рЃћрЃЎрЃарЃљрЃюрЃўрЃА **рЃЏрЃЌрЃџрЃўрЃљрЃю** рЃљрЃарЃћрЃА (Header-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ) рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃ«рЃљрЃбрЃљрЃЋрЃА Header-рЃА.
- **`esp_err_t write_text(context, line_num, format, ...);`**  
  РюЇ№ИЈ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃБрЃџ рЃбрЃћрЃЦрЃАрЃбрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџ рЃ«рЃљрЃќрЃќрЃћ **рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃљрЃарЃћрЃерЃў** (рЃърЃљрЃбрЃљрЃарЃљ, 6x8 рЃцрЃЮрЃюрЃбрЃўрЃЌ).

## 5. ­Ъџђ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ

1. **рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ (Splash Screen):**
    - рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА, рЃћрЃЎрЃарЃљрЃюрЃќрЃћ 5 рЃгрЃљрЃЏрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃЊрЃўрЃЊрЃў, рЃфрЃћрЃюрЃбрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў `init_text`.
2. **рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃарЃћрЃЪрЃўрЃЏрЃў:**
    - 5 рЃгрЃљрЃЏрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃћрЃЎрЃарЃљрЃюрЃў рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃљ рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў.
    - рЃќрЃћрЃЊрЃљ рЃќрЃЮрЃџрЃерЃў (Header) рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ "synapse" рЃџрЃЮрЃњрЃЮ рЃЊрЃљ WiFi рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃўрЃЎрЃЮрЃюрЃЎрЃљ.
    - WiFi рЃўрЃЎрЃЮрЃюрЃЎрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ рЃДрЃЮрЃЋрЃћрЃџ 5 рЃгрЃљрЃЏрЃерЃў.
3. **рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ:**
    - `FMW_EVENT_AGGREGATED_SENSOR_REPORT` рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃўрЃАрЃљрЃА, рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃљрЃарЃћрЃерЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў.
4. **API рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃћрЃЉрЃў:**
    - рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ, рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЊрЃарЃЮрЃА рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃЮрЃю `clear` рЃљрЃю `write_text` рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў, рЃарЃљрЃЌрЃљ рЃЏрЃљрЃарЃЌрЃЮрЃю рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў.

## 6. ­ЪњА рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃЮ рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃА рЃўрЃЊрЃћрЃћрЃЉрЃў

- **рЃЏрЃћрЃюрЃўрЃБрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ:** рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃарЃБрЃџрЃцрЃљрЃАрЃЮрЃЋрЃљрЃюрЃў, рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЏрЃћрЃюрЃўрЃБрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.
- **рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў:** рЃ«рЃљрЃќрЃћрЃЉрЃўрЃА, рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃцрЃўрЃњрЃБрЃарЃћрЃЉрЃўрЃА рЃ«рЃљрЃбрЃЋрЃўрЃА API-рЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ.
- **рЃЏрЃарЃљрЃЋрЃљрЃџрЃў "рЃњрЃЋрЃћрЃарЃЊрЃў":** рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃўрЃЌ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ "рЃњрЃЋрЃћрЃарЃЊрЃА" (рЃЏрЃљрЃњ., рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃў, рЃљрЃЦрЃбрЃБрЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ, рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ) рЃерЃЮрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃарЃЌрЃЋрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљ.
