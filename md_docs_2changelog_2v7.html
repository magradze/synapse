    • рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ: synapse_system_start рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃЊрЃљрЃўрЃ«рЃЋрЃћрЃгрЃљ, рЃарЃљрЃЌрЃљ рЃўрЃюрЃћрЃЦрЃфрЃўрЃљ рЃЊрЃљ рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃЮрЃЊрЃћрЃА рЃћрЃарЃЌ, рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃБрЃџ рЃфрЃўрЃЎрЃџрЃерЃў, init_level-рЃўрЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЊрЃљрЃфрЃЋрЃўрЃЌ.

­ЪљЏ рЃерЃћрЃАрЃгрЃЮрЃарЃћрЃЉрЃћрЃЉрЃў (Bug Fixes)

  • fix(core): ­Ъљъ рЃљрЃдрЃЏрЃЮрЃўрЃцрЃ«рЃЋрЃарЃљ рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃў Guru Meditation Error (LoadProhibited), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃгрЃЋрЃћрЃБрЃџрЃў рЃўрЃДрЃЮ System Manager-рЃўрЃА рЃЏрЃўрЃћрЃа рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃа рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ рЃгрЃћрЃарЃўрЃЌ. рЃљрЃ«рЃљрЃџрЃў, dependency_map-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃћрЃЦрЃфрЃўрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў рЃљрЃЏ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљрЃА рЃАрЃарЃБрЃџрЃљрЃЊ рЃљрЃњрЃЋрЃљрЃарЃћрЃЉрЃА.
  • fix(modules): ­ЪћД рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљрЃА (button_input, relay_actuator, rgb_led_indicator рЃЊрЃљ рЃљ.рЃе.), рЃарЃљрЃЌрЃљ рЃЏрЃљрЃЌ рЃЏрЃЮрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃю dependency_map-рЃўрЃА рЃЊрЃћрЃЎрЃџрЃљрЃарЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю рЃўрЃюрЃћрЃЦрЃфрЃўрЃарЃћрЃЉрЃБрЃџрЃў handle-рЃћрЃЉрЃў.

­ЪЊё рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ (Documentation)

  • docs(core): РюЇ№ИЈ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃљ base_module.h рЃЊрЃљ system_manager.c рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃљрЃ«рЃљрЃџрЃў dependency_map рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃАрЃљ рЃЊрЃљ рЃўрЃюрЃћрЃЦрЃфрЃўрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў Doxygen рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃЌ.
  • docs(design): РюЇ№ИЈ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃљ dependency_injection.md рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў, рЃарЃљрЃЌрЃљ рЃљрЃАрЃљрЃ«рЃљрЃЋрЃЊрЃћрЃА рЃљрЃ«рЃљрЃџ, offsetof()-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџ, рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљрЃА.

­ЪњЦ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў (Breaking Changes & Refactoring)

  • refactor(core): ­ЪЈЏ№ИЈ рЃњрЃљрЃюрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЊрЃљ рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА рЃЉрЃўрЃарЃЌрЃЋрЃўрЃА API-рЃА рЃАрЃарЃБрЃџрЃў рЃарЃћрЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃЊрЃћрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ, рЃерЃћрЃЏрЃфрЃўрЃарЃЊрЃћрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃЊрЃћрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃгрЃћрЃарЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў.
    • **рЃцрЃљрЃАрЃљрЃЊрЃўрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў (synapse.h):** рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ рЃћрЃарЃЌрЃўрЃљрЃюрЃў, рЃАрЃљрЃ»рЃљрЃарЃЮ рЃ░рЃћрЃЊрЃћрЃа рЃцрЃљрЃўрЃџрЃў **synapse.h**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА core рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃ»рЃљрЃарЃЮ API-рЃА (Event Bus, Service Locator, Config Manager рЃЊрЃљ рЃљ.рЃе.). рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃљрЃ«рЃџрЃљ рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў #include "synapse.h" рЃЊрЃўрЃарЃћрЃЦрЃбрЃўрЃЋрЃўрЃЌ, рЃарЃљрЃф рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃюрЃљрЃЊ рЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃА рЃЎрЃЮрЃЊрЃА.
    • рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃћрЃЉрЃўрЃА рЃарЃћрЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃљ:
      • **core рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃерЃў рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ:** рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЦрЃўрЃА рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў "рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃћрЃЉрЃў" (base_module.h, service_types.h, framework_events.h, event_payloads.h) рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃљрЃЊ рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃџ рЃўрЃЦрЃюрЃљ core рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃерЃў, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃАрЃўрЃюрЃў рЃЏрЃўрЃАрЃў API-рЃА рЃњрЃљрЃюрЃБрЃДрЃЮрЃцрЃћрЃџрЃў рЃюрЃљрЃгрЃўрЃџрЃўрЃљ.
      • **interfaces рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃА рЃарЃЮрЃџрЃўрЃА рЃЊрЃљрЃќрЃБрЃАрЃбрЃћрЃЉрЃљ:** interfaces рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў рЃљрЃ«рЃџрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃћрЃа рЃЏрЃЮрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃљрЃЉрЃАрЃбрЃљрЃЦрЃбрЃБрЃџ рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃћрЃЉрЃА (relay_interface.h, display_driver_interface.h рЃЊрЃљ рЃљ.рЃе.), рЃарЃљрЃф рЃЦрЃЏрЃюрЃўрЃА рЃБрЃцрЃарЃЮ рЃЏрЃЎрЃљрЃцрЃўрЃЮ рЃЊрЃљ рЃџрЃЮрЃњрЃўрЃЎрЃБрЃа рЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃА.
