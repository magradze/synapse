Detailed Description

Manages menu state, navigation, and rendering for the UI Manager.

This header declares the core functions responsible for all menu-related logic within the UI system. This includes handling user navigation input (up, down, ok), rendering the appropriate menu screen based on the current UI state, and populating the dynamic list of modules.
