РЈ▒№ИЈ рЃЏрЃЮрЃЊрЃБрЃџрЃў <tt>system_timer</tt>

1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў: system_timer
рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ: utils
рЃЋрЃћрЃарЃАрЃўрЃљ: 1.0.0
рЃљрЃЋрЃбрЃЮрЃарЃў: Synapse Framework Team

system_timer рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃў, рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЊрЃарЃЮрЃќрЃћ-рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљрЃА. рЃўрЃА рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА ESP-IDF-рЃўрЃА рЃЏрЃљрЃдрЃљрЃџрЃў рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃўрЃА **esp_timer** рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃА рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљрЃА.

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃњрЃљрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃЮрЃА рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃў рЃљрЃю рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ. рЃўрЃЏрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ, рЃарЃЮрЃЏ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃЏрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃЏрЃљ рЃЌрЃљрЃЋрЃљрЃЊ рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЮрЃА FreeRTOS рЃбрЃљрЃўрЃЏрЃћрЃарЃћрЃЉрЃў, рЃЏрЃљрЃЌ рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ, рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ, "рЃАрЃЌрЃ«рЃЮрЃЋрЃЮрЃю" system_timer-рЃА, рЃарЃЮрЃЏ ЯдеЯд┐Яд░ЯДЇЯддЯд┐ЯдиЯДЇЯдЪ рЃЊрЃарЃЮрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ Event Bus-рЃќрЃћ рЃњрЃљрЃЏрЃЮрЃљрЃЦрЃЋрЃћрЃДрЃюрЃЮрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃў.

2. ­ЪЈЌ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

  • рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ-рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў (Service-driven): рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА рЃАрЃБрЃцрЃЌрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃџрЃў. рЃўрЃА рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА timer_api_t рЃАрЃћрЃарЃЋрЃўрЃАрЃА Service Locator-рЃерЃў рЃЊрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃќрЃћ.
  • рЃЊрЃћрЃЎрЃљрЃърЃџрЃўрЃюрЃњрЃў (Decoupling): рЃћрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃ«рЃћрЃџрЃА рЃБрЃгрЃДрЃЮрЃЉрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃа рЃерЃћрЃЏрЃфрЃўрЃарЃћрЃЉрЃљрЃА. рЃЏрЃЮрЃЊрЃБрЃџрЃА, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃърЃћрЃарЃўрЃЮрЃЊрЃБрЃџрЃў рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ, рЃљрЃдрЃљрЃа рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ callback рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃљ. рЃўрЃА рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃўрЃЌрЃ«рЃЮрЃЋрЃА рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљрЃА рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЌрЃљрЃЋрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ handle_event рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃерЃў рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃЏрЃљрЃА.
  • рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљ: рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА esp_timer-рЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃцрЃарЃЮ рЃЏрЃАрЃБрЃЉрЃБрЃЦрЃў рЃЊрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ FreeRTOS рЃбрЃљрЃўрЃЏрЃћрЃарЃћрЃЉрЃў, рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЊрЃўрЃЊрЃў рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃўрЃЌ рЃбрЃљрЃўрЃЏрЃћрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў. рЃўрЃА рЃљрЃАрЃћрЃЋрЃћ рЃќрЃдрЃБрЃЊрЃљрЃЋрЃА рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃбрЃљрЃўрЃЏрЃћрЃарЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃА (CONFIG_SYSTEM_TIMER_MAX_TIMERS), рЃарЃљрЃф рЃўрЃфрЃљрЃЋрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃњрЃљрЃю.
  • **рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃћ (init_level: 35):** рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃЉрЃўрЃарЃЌрЃЋрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃБрЃЏрЃћрЃбрЃћрЃАрЃЮрЃЉрЃљрЃЏрЃЊрЃћ, рЃарЃљрЃЌрЃљ рЃЏрЃљрЃЌ рЃерЃћрЃФрЃџрЃЮрЃю рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃљрЃюрЃЌ start рЃцрЃљрЃќрЃљрЃерЃў.

3. РџЎ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ Kconfig-рЃўрЃА (idf.py menuconfig) рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

**Kconfig рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:**

Component config -> Synapse Framework -> Modules -> System Timer Service рЃЏрЃћрЃюрЃўрЃБрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ:

  • **Maximum number of concurrent timers:** рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃбрЃљрЃўрЃЏрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЊрЃљрЃњрЃћрЃњрЃЏрЃўрЃџрЃў рЃўрЃЋрЃћрЃюрЃЌрЃў рЃўрЃЎрЃљрЃЋрЃћрЃЉрЃА рЃћрЃарЃЌ рЃАрЃџрЃЮрЃбрЃА.

**system_config.json рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

{
"type": "system_timer",
"enabled": true,
"config": {
"instance_name": "main_timer_service"
}
}

4. ­ЪЏа№ИЈ Service API (<tt>timer_api_t</tt>)

рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃА рЃбрЃўрЃърЃўрЃЌ FMW_SERVICE_TYPE_TIMER_API.

API рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:

  • **fmw_timer_handle_t schedule_event(const char* event_name, uint32_t interval_ms, bool is_periodic);**
    ­ЪЌЊ№ИЈ рЃњрЃћрЃњрЃЏрЃљрЃЋрЃА рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃљрЃА.
    • event_name: рЃўрЃЋрЃћрЃюрЃЌрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃЊрЃћрЃА. рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ: рЃћрЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃърЃћрЃарЃЏрЃљрЃюрЃћрЃюрЃбрЃБрЃџрЃў (рЃЏрЃљрЃњ., #define рЃљрЃю static const char*).
