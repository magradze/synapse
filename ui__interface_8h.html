Detailed Description

Defines the public API and contracts for the Modular UI system.

Author
Giorgi Magradze
Version
1.0.0
Date
2025-09-12

This file establishes the "contract" between the central UI Manager and any module that wishes to provide a UI component. It defines the structures and function pointers that modules must implement to register and manage their UI elements dynamically.
