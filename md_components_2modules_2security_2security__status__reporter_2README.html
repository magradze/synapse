1. ­ЪЊю рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

security_status_reporter рЃљрЃарЃўрЃА Synapse Framework-рЃўрЃА рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў. рЃЏрЃўрЃАрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃљ, рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃБрЃџрЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџ рЃАрЃбрЃљрЃбрЃБрЃАрЃћрЃЉрЃќрЃћ: Secure Boot-рЃўрЃА рЃЊрЃљ Flash Encryption-рЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљрЃќрЃћ.

рЃћрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА рЃћрЃарЃЌрЃ«рЃћрЃџ рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃљрЃЏ рЃАрЃбрЃљрЃбрЃБрЃАрЃћрЃЉрЃА, рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА (рЃљрЃЎрЃћрЃерЃўрЃарЃћрЃЉрЃА) рЃЏрЃљрЃЌ рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ, Service Locator-рЃўрЃАрЃљ рЃЊрЃљ Command Router-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ, рЃљрЃгрЃЋрЃЊрЃўрЃА рЃљрЃЏ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃА, рЃўрЃАрЃћ рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА. рЃћрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃўрЃЏрЃўрЃА рЃЊрЃљрЃАрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃџрЃљрЃЊ, рЃарЃЮрЃЏ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЊрЃљрЃфрЃБрЃџ, production-ready рЃарЃћрЃЪрЃўрЃЏрЃерЃў.

2. ­ЪЈЏ№ИЈ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

  • Service-driven: рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃўрЃА рЃАрЃБрЃцрЃЌрЃљ "рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃА рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃџрЃў". рЃўрЃА рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА security_status_api_t рЃАрЃћрЃарЃЋрЃўрЃАрЃА Service Locator-рЃерЃў, рЃарЃљрЃЌрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃЏрЃљ (рЃЏрЃљрЃњ., selftest_manager) рЃерЃћрЃФрЃџрЃЮрЃю рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃљрЃЊ рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ.
  • **рЃљрЃЊрЃарЃћрЃБрЃџрЃў рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ (init_level: 22):** рЃарЃЮрЃњрЃЮрЃарЃф рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃЊрЃарЃћрЃБрЃџ рЃћрЃбрЃљрЃърЃќрЃћ, рЃарЃљрЃЌрЃљ рЃЏрЃўрЃАрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃўрЃДрЃЮрЃА рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
  • рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЎрЃћрЃерЃўрЃарЃћрЃЉрЃљ: рЃАрЃбрЃљрЃбрЃБрЃАрЃћрЃЉрЃў рЃўрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃ«рЃћрЃџ, init рЃцрЃљрЃќрЃљрЃерЃў. рЃћрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃћрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў runtime-рЃерЃў рЃљрЃа рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ eFuse-рЃћрЃЉрЃўрЃЊрЃљрЃю рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃюрЃћрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљрЃљ.
  • CLI рЃЊрЃљ MQTT рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ: рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА security status рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА. рЃљрЃЏ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃАрЃљрЃА, рЃўрЃА рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЎрЃЮрЃюрЃАрЃЮрЃџрЃерЃў, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃљрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃА FMW_EVENT_SECURITY_STATUS_READY рЃўрЃЋрЃћрЃюрЃЌрЃА, рЃарЃљрЃф mqtt_manager-рЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА, рЃћрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ MQTT-рЃўрЃЌ рЃњрЃљрЃљрЃЋрЃарЃфрЃћрЃџрЃЮрЃА.
