Detailed Description

RGB LED Indicator Service API structure.

This is the "contract" that the rgb_led_indicator module implements and other modules (like ota_manager) consume.

◆ release_control

esp_err_t(* rgb_led_api_t::release_control) (void)

Releases manual control of the LED.

Returns the LED to its automatic, event-driven state, where it reflects the overall system status (e.g., WiFi connection).

Returns
ESP_OK on success.

esp_err_t(* rgb_led_api_t::set_color) (uint8_t r, uint8_t g, uint8_t b)

Sets a static color on the LED.

This action overrides any active effect (blink, pulse) and takes manual control over the LED.

Parameters
