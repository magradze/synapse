# рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃЎрЃЮрЃюрЃЋрЃћрЃюрЃфрЃўрЃљ

**рЃАрЃбрЃљрЃбрЃБрЃАрЃў:** рЃБрЃЏрЃЎрЃљрЃфрЃарЃћрЃАрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ (Mandatory)
**рЃЋрЃћрЃарЃАрЃўрЃљ:** 2.0.0

## 1. ­ЪЊю рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃў

**рЃДрЃЋрЃћрЃџрЃљ runtime рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа `config.json` рЃцрЃљрЃўрЃџрЃерЃў. рЃЎрЃЮрЃЊрЃерЃў `hardcoded` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЊрЃљрЃБрЃерЃЋрЃћрЃЉрЃћрЃџрЃўрЃљ.**

## 2. ­ЪЊѓ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃгрЃДрЃљрЃарЃЮрЃћрЃЉрЃў рЃЊрЃљ рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃљ

Synapse Framework-рЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЏрЃарЃљрЃЋрЃљрЃџрЃерЃарЃўрЃљрЃю, рЃЊрЃћрЃфрЃћрЃюрЃбрЃарЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃА. рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃњрЃљрЃюрЃўрЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (рЃЏрЃљрЃдрЃџрЃўрЃЊрЃљрЃю рЃЊрЃљрЃЉрЃџрЃўрЃАрЃЎрЃћрЃю):

1. **NVS (Non-Volatile Storage):**
    * **рЃарЃЮрЃџрЃў:** рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ, рЃАрЃљрЃЊрЃљрЃф рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ, рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃфрЃЋрЃџрЃўрЃџрЃў рЃљрЃю `provisioning`-рЃўрЃЌ рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ. NVS-рЃерЃў рЃерЃћрЃюрЃљрЃ«рЃБрЃџрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ **рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃдрЃљрЃџрЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃўрЃАрЃљрЃљ** рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃљрЃгрЃћрЃарЃА рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃюрЃА.

2. **`{module_name}/config.json` (Runtime Default-рЃћрЃЉрЃў):**
    * **рЃарЃЮрЃџрЃў:** рЃћрЃА рЃљрЃарЃўрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА **default runtime рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ**. рЃљрЃЦ рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃЏрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃЏрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЊрЃљрЃљрЃгрЃћрЃарЃЮрЃА `Kconfig`-рЃўрЃА `default`-рЃћрЃЉрЃў.

3. **`Kconfig` (Build-рЃЊрЃарЃЮрЃўрЃА Default-рЃћрЃЉрЃў):**
    * **рЃарЃЮрЃџрЃў:** рЃћрЃА рЃљрЃарЃўрЃА **"рЃАрЃўрЃЏрЃљрЃарЃЌрЃџрЃўрЃА рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃгрЃДрЃљрЃарЃЮ"** default рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃгрЃДрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф firmware-рЃерЃў рЃЕрЃљрЃўрЃерЃћрЃюрЃћрЃЉрЃљ.

4. **`configs/system_config.json` (рЃЉрЃљрЃќрЃўрЃАрЃБрЃарЃў):**
    * **рЃарЃЮрЃџрЃў:** рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃа рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА (`firmware`, `global_config`) рЃЊрЃљ `modules` рЃЏрЃљрЃАрЃўрЃЋрЃўрЃА рЃАрЃљрЃгрЃДрЃўрЃА, рЃфрЃљрЃарЃўрЃћрЃџ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА.

## 3. ­ЪЈЌ№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ, рЃарЃЮрЃЏ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃљрЃЊ рЃЊрЃљрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ, `Config Manager` runtime-рЃерЃў рЃљрЃгрЃДрЃЮрЃЉрЃА рЃћрЃарЃЌрЃўрЃљрЃю, рЃџрЃЮрЃњрЃўрЃЎрЃБрЃа рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў (`relay_actuator/config.json`):**

```json
[
  {
    "type": "relay_actuator",
    "enabled": true,
    "config": {
        "instance_name": "main_light",
        "pin": 23,
        "active_level": 0
    }
  }
]
```

## 4. ­ЪЊЮ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃАрЃўрЃюрЃњрЃўрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў

рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА `_create` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃўрЃдрЃћрЃЉрЃА рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃАрЃарЃБрЃџ JSON рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА. рЃљрЃЏ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃЊрЃљрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃџрЃљрЃЊ, рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў, 3-рЃюрЃљрЃЉрЃўрЃ»рЃўрЃљрЃюрЃў рЃърЃљрЃбрЃћрЃарЃюрЃў `parse_config` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃерЃў.

### `parse_config` рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА 3-рЃюрЃљрЃЉрЃўрЃ»рЃўрЃљрЃюрЃў рЃџрЃЮрЃњрЃўрЃЎрЃљ

1. **рЃюрЃљрЃЉрЃўрЃ»рЃў 1: Default рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ:** рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃў, `private_data` рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃА рЃЏрЃўрЃљрЃюрЃўрЃГрЃћрЃЌ `default` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў, рЃарЃЮрЃњрЃЮрЃарЃф рЃгрЃћрЃАрЃў, `Kconfig`-рЃўрЃЊрЃљрЃю.
2. **рЃюрЃљрЃЉрЃўрЃ»рЃў 2: JSON-рЃўрЃЊрЃљрЃю рЃњрЃљрЃЊрЃљрЃгрЃћрЃарЃљ:** рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `synapse_utils.h`-рЃерЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў (`synapse_config_get_string_from_node`, `..._get_int_...` рЃЊрЃљ рЃљ.рЃе.), рЃарЃљрЃЌрЃљ рЃАрЃфрЃљрЃЊрЃЮрЃЌ `default` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃћрЃарЃљ `config.json`-рЃўрЃЊрЃљрЃю рЃЏрЃЮрЃАрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃЌ.
3. **рЃюрЃљрЃЉрЃўрЃ»рЃў 3: рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ:** рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЉрЃЮрЃџрЃЮрЃА, рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ **рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ** рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃЏрЃљ рЃЏрЃўрЃўрЃдрЃЮ рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ. рЃЌрЃБ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃљрЃа рЃљрЃарЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў, рЃЊрЃљрЃљрЃЉрЃарЃБрЃюрЃћрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ.

### `parse_config` рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

```c
// health_monitor.c-рЃерЃў
static esp_err_t parse_config(const cJSON *config_node, health_monitor_private_data_t *private_data) {
    // рЃюрЃљрЃЉрЃўрЃ»рЃў 1: Default рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ Kconfig-рЃўрЃЊрЃљрЃю
    private_data->check_interval_sec = CONFIG_HEALTH_MONITOR_DEFAULT_INTERVAL;
    private_data->min_free_heap_kb = CONFIG_HEALTH_MONITOR_DEFAULT_HEAP_KB;

    // рЃюрЃљрЃЉрЃўрЃ»рЃў 2: JSON-рЃўрЃЊрЃљрЃю рЃњрЃљрЃЊрЃљрЃгрЃћрЃарЃўрЃА рЃЏрЃфрЃЊрЃћрЃџрЃЮрЃЉрЃљ synapse_utils-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ
    synapse_config_get_int_from_node(TAG, config_node, "check_interval_sec", (int*)&private_data->check_interval_sec);
    
    const cJSON *thresholds = cJSON_GetObjectItem(config_node, "thresholds");
    if (thresholds) {
        synapse_config_get_int_from_node(TAG, thresholds, "min_free_heap_kb", (int*)&private_data->min_free_heap_kb);
    }

    // рЃюрЃљрЃЉрЃўрЃ»рЃў 3: рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ (рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ)
    ESP_LOGI(TAG, "Health Monitor configured: Interval=%us, Min Heap=%ukB", 
             private_data->check_interval_sec, private_data->min_free_heap_kb);

    return ESP_OK;
}
```

## 5. ­Ъїљ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃЊрЃљрЃю

рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃЏрЃЮрЃ«рЃЊрЃћрЃА **рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ** `Config Manager`-рЃўрЃА API-рЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ, **рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃў рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃўрЃА (dot-notation)** рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

* **рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃАрЃЌрЃЋрЃўрЃА:** `"global_config.parameter_name"`
* **рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃАрЃЌрЃЋрЃўрЃА:** `"instance_name.parameter_name"`

```c
// `main_light` рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃўрЃА `pin` рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
int relay_pin;
synapse_config_get_int("main_light.pin", &relay_pin);
```

## 6. РЮї рЃљрЃЎрЃарЃФрЃљрЃџрЃБрЃџрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў

* **рЃљрЃарЃљрЃАрЃЊрЃарЃЮрЃА:**
  * рЃЎрЃЮрЃЊрЃерЃў `hardcoded` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.
  * `parse_config` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃерЃў `cJSON_*` рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ (рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `synapse_utils`).
  * `module.json`-рЃерЃў runtime рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ.
  * `system_config.json`-рЃерЃў `modules` рЃЏрЃљрЃАрЃўрЃЋрЃўрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃерЃћрЃЋрЃАрЃћрЃЉрЃљ.

## 7. РюЁ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў

* **рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА:**
  * рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `synapse_utils.h`-рЃўрЃА рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃАрЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА.
  * рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЌ `default` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў `Kconfig`-рЃерЃў.
  * рЃљрЃдрЃгрЃћрЃарЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА `README.md` рЃцрЃљрЃўрЃџрЃерЃў.
  * рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `synapse_config_get_*` API-рЃћрЃЉрЃў рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА.
  * рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЌ рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА default рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃЏрЃўрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа `config.json` рЃцрЃљрЃўрЃџрЃерЃў.
