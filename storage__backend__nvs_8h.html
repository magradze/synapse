esp_err_t storage_backend_nvs_init ( storage_api_t *  api_out)

Initializes the NVS backend and populates the API function table.

This function performs the necessary setup for using NVS as the storage medium and assigns the NVS-specific implementation functions to the provided API structure.

Parameters
[out]api_outA pointer to the storage_api_t structure that will be filled with function pointers for NVS operations.
Returns
  • ESP_OK: If the NVS backend was initialized successfully.
  • ESP_FAIL: If a critical error occurred during initialization.
