Detailed Description

Intelligent watchdog timer management with heartbeat monitoring.

Author
Synapse Framework Team
Version
1.0.0
Date
2025-07-02

This module implements an application-aware watchdog. It subscribes the main task to the ESP-IDF Task Watchdog Timer (TWDT) and only "feeds" it if all registered critical modules send a periodic heartbeat, ensuring the entire system is responsive, not just a single task.
